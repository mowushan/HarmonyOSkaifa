🧩 知识讲解 · 数据库让数据“住下来”

1. 为什么需要数据库？—— 对比小剧场

📊 情景对比：

  • 📝 变量存储：App一关闭，数据就消失（就像写在黑板上，下课就擦掉）。
  • 📄 文件存储：能保存，但查找一条记录要读整个文件（慢）。
  • 🗄️ 数据库存储：结构化管理，用SQL语句“秒查”数据，还能多表关联。

🎯 我们的资产管理系统需要： 存储几十上百条资产，支持按名称搜索、按价格排序 → 必须用数据库！

2. SQLite —— 鸿蒙内置的轻量级数据库

SQLite是鸿蒙系统自带的嵌入式数据库，不需要单独安装，一个文件就是一个数据库，非常适合移动应用。

3. CRUD · 数据库的四大基本功

📝 CREATE（增）
INSERT INTO asset (name, price, quantity) VALUES ('笔记本电脑', 5999, 10);
🔍 RETRIEVE（查）
SELECT * FROM asset WHERE price > 1000;
✏️ UPDATE（改）
UPDATE asset SET price = 5499 WHERE name = '笔记本电脑';
🗑️ DELETE（删）
DELETE FROM asset WHERE id = 5;

    • import relationalStore from '@ohos.data.relationalStore';
import { BusinessError } from '@ohos.base';

const SQL_CREATE_TABLE = `
    CREATE TABLE IF NOT EXISTS asset (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT NOT NULL,
        price REAL NOT NULL,
        quantity INTEGER NOT NULL,
        createTime TEXT DEFAULT (datetime('now', 'localtime'))
    )
`;
  • 获取数据库实例：编写初始化函数
    • let rdbStore: relationalStore.RdbStore | null = null;

export async function initDatabase(): Promise<void> {
    const config: relationalStore.StoreConfig = {
        name: 'AssetDB.db',
        securityLevel: relationalStore.SecurityLevel.S1
    };
    try {
        rdbStore = await relationalStore.getRdbStore(config);
        await rdbStore.executeSql(SQL_CREATE_TABLE);
        console.info('数据库初始化成功，资产表已创建');
    } catch (err) {
        let error = err as BusinessError;
        console.error(`数据库初始化失败: ${error.message}`);
    }
}

export async function insertAsset(asset: Asset): Promise<number> {
    if (!rdbStore) {
        throw new Error('数据库未初始化');
    }
    const valueBucket: relationalStore.ValuesBucket = {
        name: asset.name,
        price: asset.price,
        quantity: asset.quantity
    };
    const rowId = await rdbStore.insert('asset', valueBucket);
    console.info(`插入成功，行ID: ${rowId}`);
    return rowId;
}

    🔧 实训三：查询数据（SELECT）- 展示资产列表（20分钟）

  • 添加查询所有资产的函数
    • export async function queryAllAssets(): Promise<Asset[]> {
    if (!rdbStore) {
        throw new Error('数据库未初始化');
    }
    const predicates = new relationalStore.RdbPredicates('asset');
    const resultSet = await rdbStore.query(predicates);
    const assets: Asset[] = [];
    while (resultSet.goToNextRow()) {
        const id = resultSet.getLong(resultSet.getColumnIndex('id'));
        const name = resultSet.getString(resultSet.getColumnIndex('name'));
        const price = resultSet.getDouble(resultSet.getColumnIndex('price'));
        const quantity = resultSet.getLong(resultSet.getColumnIndex('quantity'));
        assets.push({ id, name, price, quantity });
    }
    resultSet.close();
    return assets;
}
  • 修改Asset接口：增加可选id字段
    • export interface Asset {
    id?: number;
    name: string;
    price: number;
    quantity: number;
}
  • 在页面中显示：用List组件展示查询结果

async loadData() {
    this.assetList = await queryAllAssets();
}

build() {
    List() {
        ForEach(this.assetList, (item: Asset) => {
            ListItem() {
                Text(`${item.name}  |  ¥${item.price}  |  数量:${item.quantity}`)
                    .padding(12)
            }
        })
    }
}

    🔧 实训四：更新和删除（UPDATE/DELETE）- 修改资产（15分钟）

  • 添加更新价格函数
    • export async function updateAssetPrice(id: number, newPrice: number): Promise<number> {
    if (!rdbStore) throw new Error('数据库未初始化');
    const valueBucket: relationalStore.ValuesBucket = { price: newPrice };
    const predicates = new relationalStore.RdbPredicates('asset');
    predicates.equalTo('id', id);
    const changedRows = await rdbStore.update(valueBucket, predicates);
    console.info(`更新了 ${changedRows} 行`);
    return changedRows;
}

export async function deleteAssetById(id: number): Promise<number> {
    if (!rdbStore) throw new Error('数据库未初始化');
    const predicates = new relationalStore.RdbPredicates('asset');
    predicates.equalTo('id', id);
    const deletedRows = await rdbStore.delete(predicates);
    console.info(`删除了 ${deletedRows} 行`);
    return deletedRows;
}
  • 给列表项添加按钮：每个资产旁边放“修改价格”和“删除”按钮
    • Button('降价100')

        this.loadData();  // 刷新列表
    })
Button('删除')
    .onClick(async () => {
        await deleteAssetById(item.id!);
        this.loadData();
    })
