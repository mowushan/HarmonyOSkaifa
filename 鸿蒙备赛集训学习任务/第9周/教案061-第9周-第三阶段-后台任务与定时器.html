🧩 知识讲解 · 后台任务与定时器

1. 为什么需要后台任务？—— 应用不能“睡着”

💤 趣味引入： 如果应用是一个员工，他不能只在老板看着的时候干活，老板一走就摸鱼。后台任务就是让应用“偷偷”在后台干活的能力。

应用场景举例：

  • 📱 资产管理App：每隔1小时自动同步云端数据
  • ⏰ 闹钟/提醒：到点弹出通知
  • 📡 消息推送：实时接收新消息
  • 🧹 清理缓存：定期删除过期文件

鸿蒙中的后台任务分类：

  • 短时任务： 几秒内完成（如网络请求）
  • 长时任务： 需要持续运行（需申请权限，如导航、音乐播放）
  • 定时任务： 周期性执行（我们今天的重点！）

2. 鸿蒙中的定时器 —— setInterval / setTimeout

ArkTS 基于 TypeScript，支持 JavaScript 的定时器函数：

  • setTimeout(回调, 延迟毫秒)：延迟一段时间后执行一次。
  • setInterval(回调, 间隔毫秒)：每隔一段时间重复执行。
  • clearInterval(定时器ID) / clearTimeout(定时器ID)：停止定时器。

⚠️ 注意： 组件销毁时务必清除定时器，否则会造成内存泄漏和性能问题！

// 示例1：延迟2秒后执行一次
let timeoutId = setTimeout(() => {
    console.info("2秒后执行，只执行一次");
}, 2000);

// 示例2：每隔1秒重复执行
let intervalId = setInterval(() => {
    console.info("每1秒执行一次");
}, 1000);

// 示例3：停止定时器（重要！）
clearInterval(intervalId);
clearTimeout(timeoutId);

3. 后台任务与页面生命周期

当用户离开页面时，定时器可能仍在运行，这会浪费电量。我们需要在页面销毁时（`aboutToDisappear`）清理定时器。

@Entry
@Component
struct TimerDemo {
    private intervalId: number = -1;

    aboutToAppear() {
        // 页面出现时启动定时器
        this.intervalId = setInterval(() => {
            console.info("后台任务执行中...");
        }, 5000);
    }

    aboutToDisappear() {
        // 页面销毁时清理定时器，防止内存泄漏
        if (this.intervalId !== -1) {
            clearInterval(this.intervalId);
        }
    }

    build() {
        Column() {
            Text('定时器演示页面')
                .fontSize(20)
                .margin(20)
            Text('打开控制台，每5秒输出日志')
                .fontSize(14)
                .fontColor('#666')
        }
        .width('100%')
        .height('100%')
    }
}

4. 后台任务的限制（重要！）

🔋 鸿蒙系统为了省电，对后台任务有严格限制：

  • 应用退到后台几分钟后，定时器会被冻结（不再执行）。
  • 如果需要长时间后台运行（如音乐播放），需使用 长时任务（BackgroundTasks）并申请权限。
  • 我们的资产管理系统： 不需要长时间后台，只需要在应用前台运行时定期同步数据即可。

✅ 今天的实训只关注：应用在前台或未退出时，如何实现周期性任务。

⚙️ 技能实训 · 动手让定时器跑起来

✅ 实训目标：在VSCode/DevEco Studio中编写定时器Demo，并学会停止定时器

🔧 实训一：新建项目并编写定时器页面（15分钟）

  • 在DevEco Studio中新建一个空白项目（或使用之前的集训项目），在`entry/src/main/ets/pages`下新建一个页面 `TimerPage.ets`。
  • 编写基础页面框架： 包含一个显示计数的Text，一个开始/停止按钮。
  • 引入状态变量： 使用 `@State count: number = 0` 记录定时器执行次数。
  • 实现startTimer函数，用setInterval每秒更新count。
  • 实现stopTimer函数，清除定时器并将count归零。
    • @Entry
@Component
struct TimerPage {
    @State count: number = 0;
    private intervalId: number = -1;

    startTimer() {
        if (this.intervalId !== -1) return;
        this.intervalId = setInterval(() => {
            this.count++;
            console.info(`定时器执行了 ${this.count} 次`);
        }, 1000);
    }

    stopTimer() {
        if (this.intervalId !== -1) {
            clearInterval(this.intervalId);
            this.intervalId = -1;
            this.count = 0;
            console.info("定时器已停止");
        }
    }

    aboutToDisappear() {
        this.stopTimer();
    }

    build() {
        Column({ space: 20 }) {
            Text(`定时器已执行 ${this.count} 秒`)
                .fontSize(24)
                .fontWeight(FontWeight.Bold)
            Row({ space: 30 }) {
                Button('开始定时')
                    .onClick(() => this.startTimer())
                    .backgroundColor('#0078d4')
                Button('停止定时')
                    .onClick(() => this.stopTimer())
                    .backgroundColor('#dc3545')
            }
            Text('提示：退出页面时会自动停止定时器')
                .fontSize(12)
                .fontColor('#888')
        }
        .width('100%')
        .height('100%')
        .justifyContent(FlexAlign.Center)
    }
}

    📱 运行效果： 点击“开始定时”，屏幕上的数字每秒+1，控制台输出日志；点击“停止定时”，数字归零，日志停止。

    🔧 实训二：模拟“资产自动同步”（20分钟）

  • 优化定时任务： 不只是count++，而是模拟调用同步函数 `syncAssetData()`。
  • 定义同步函数： 打印一条日志 "正在同步资产数据..."，并随机生成一个数字模拟“发现N条更新”。
  • 使用`@State`显示最后一次同步时间。
    • @Entry
@Component
struct AutoSyncDemo {
    @State lastSyncTime: string = '尚未同步';
    @State syncCount: number = 0;
    private intervalId: number = -1;

    syncAssetData() {
        this.syncCount++;
        const now = new Date();
        const timeStr = `${now.getHours()}:${now.getMinutes()}:${now.getSeconds()}`;
        this.lastSyncTime = timeStr;
        const randomUpdate = Math.floor(Math.random() * 5);
        console.info(`[自动同步] 第${this.syncCount}次同步，发现${randomUpdate}条资产更新，时间：${timeStr}`);
        // 实际项目中这里会调用网络请求或数据库操作
    }

    startAutoSync() {
        if (this.intervalId !== -1) return;
        this.syncAssetData(); // 立即执行一次
        this.intervalId = setInterval(() => {
            this.syncAssetData();
        }, 10000); // 每10秒同步一次
    }

    stopAutoSync() {
        if (this.intervalId !== -1) {
            clearInterval(this.intervalId);
            this.intervalId = -1;
            console.info("自动同步已停止");
        }
    }

    aboutToDisappear() {
        this.stopAutoSync();
    }

    build() {
        Column({ space: 16 }) {
            Text('⏰ 资产自动同步模拟器')
                .fontSize(22)
                .fontWeight(FontWeight.Bold)
            Text(`上次同步时间：${this.lastSyncTime}`)
                .fontSize(18)
                .fontColor('#0066cc')
            Text(`已同步次数：${this.syncCount}`)
                .fontSize(16)
            Row({ space: 20 }) {
                Button('启动自动同步')
                    .onClick(() => this.startAutoSync())
                    .backgroundColor('#28a745')
                Button('停止同步')
                    .onClick(() => this.stopAutoSync())
                    .backgroundColor('#dc3545')
            }
            Text('每10秒自动模拟一次资产同步，打开控制台查看详细日志')
                .fontSize(12)
                .fontColor('#666')
        }
        .width('100%')
        .height('100%')
        .justifyContent(FlexAlign.Center)
    }
}

    🔧 实训三：课堂挑战——倒计时功能（20分钟）

  • 需求： 实现一个从60秒倒计时，到0时自动停止并弹出一个提示框（或控制台输出“倒计时结束”）。
  • 技术点： 使用setInterval，每秒减少剩余秒数，当剩余秒数 ≤0 时清除定时器。
  • 界面要求： 一个大号数字显示倒计时，一个“开始倒计时”按钮，倒计时进行中不可重复开始。
    • @Entry
@Component
struct CountdownTimer {
    @State remainingSeconds: number = 60;
    @State isRunning: boolean = false;
    private intervalId: number = -1;

    startCountdown() {
        if (this.isRunning) return;
        if (this.remainingSeconds <= 0) {
            this.remainingSeconds = 60;
        }
        this.isRunning = true;
        this.intervalId = setInterval(() => {
            if (this.remainingSeconds > 0) {
                this.remainingSeconds--;
                console.info(`倒计时：${this.remainingSeconds}秒`);
            }
            if (this.remainingSeconds === 0) {
                clearInterval(this.intervalId);
                this.intervalId = -1;
                this.isRunning = false;
                console.info("⏰ 倒计时结束！");
                // 实际项目中可以弹出提示框或震动
            }
        }, 1000);
    }

    resetCountdown() {
        if (this.intervalId !== -1) {
            clearInterval(this.intervalId);
            this.intervalId = -1;
        }
        this.remainingSeconds = 60;
        this.isRunning = false;
    }

    aboutToDisappear() {
        if (this.intervalId !== -1) clearInterval(this.intervalId);
    }

    build() {
        Column({ space: 20 }) {
            Text(`${this.remainingSeconds}`)
                .fontSize(56)
                .fontWeight(FontWeight.Bold)
                .fontColor('#0078d4')
            Row({ space: 20 }) {
                Button('开始倒计时')
                    .onClick(() => this.startCountdown())
                    .enabled(!this.isRunning)
                    .backgroundColor('#0078d4')
                Button('重置')
                    .onClick(() => this.resetCountdown())
                    .backgroundColor('#ff9800')
            }
            Text(this.isRunning ? '倒计时进行中...' : '点击开始，60秒倒计时')
                .fontSize(14)
                .fontColor('#666')
        }
        .width('100%')
        .height('100%')
        .justifyContent(FlexAlign.Center)
    }
}
