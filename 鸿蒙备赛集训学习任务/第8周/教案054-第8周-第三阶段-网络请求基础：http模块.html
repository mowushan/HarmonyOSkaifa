🧩 知识讲解 · 网络请求那些事

1. HTTP协议是什么？—— 互联网的“普通话”

📦 趣味比喻： 你想从淘宝买个手机，你需要：告诉卖家“我要买”（请求），卖家把货发给你（响应）。HTTP就是电脑之间的“下单-发货”语言。

常用请求方法：

  • GET： 从服务器“拿”数据（比如获取资产列表）。
  • POST： 向服务器“提交”数据（比如新增资产）。

常见状态码（服务器给你的暗号）：

  • 200：一切OK，数据在路上了！
  • 404：你要的东西服务器上找不到（网址写错了）。
  • 500：服务器自己出问题了（不是你的错）。

2. 鸿蒙http模块 —— 三步发起请求

// 1. 导入http模块
import http from '@ohos.net.http';

// 2. 创建http请求对象
let httpRequest = http.createHttp();

// 3. 发起GET请求
httpRequest.request('https://api.example.com/assets', {
    method: http.RequestMethod.GET,
    header: { 'Content-Type': 'application/json' }
}, (err, data) => {
    if (!err && data.responseCode === 200) {
        console.info('成功获取数据：' + data.result);
    } else {
        console.error('请求失败：' + JSON.stringify(err));
    }
});

关键点： 网络请求是异步的，不会卡住界面。结果通过回调函数返回，拿到后更新UI。

3. 网络权限配置 —— 给App“上网证”

🔐 鸿蒙应用默认不能上网，需要在 module.json5 文件中申请权限：

"requestPermissions": [
  {
    "name": "ohos.permission.INTERNET"
  }
]

🔧 实训一：创建网络请求工具函数（15分钟）

  • 打开项目： 在DevEco Studio中打开“AssetManagement”项目。
  • 新建文件： 在 `entry/src/main/ets/common/` 下新建 `HttpUtil.ets`。
  • 编写工具函数： 封装一个GET请求方法。
    • // HttpUtil.ets
import http from '@ohos.net.http';

export class HttpUtil {
  // GET请求封装
  static get(url: string): Promise<string> {
    return new Promise((resolve, reject) => {
      let httpRequest = http.createHttp();
      httpRequest.request(url, {
        method: http.RequestMethod.GET,
        connectTimeout: 10000,  // 10秒超时
        readTimeout: 10000
      }, (err, data) => {
        if (!err) {
          if (data.responseCode === 200) {
            resolve(data.result);
          } else {
            reject(`HTTP错误: ${data.responseCode}`);
          }
        } else {
          reject(JSON.stringify(err));
        }
        httpRequest.destroy(); // 销毁请求对象
      });
    });
  }
}

    ✅ 老师巡导： 确保学生理解Promise的用法，强调最后要调用 httpRequest.destroy() 释放资源。

    🔧 实训二：在页面中调用网络请求（20分钟）

  • 打开 Index.ets 页面（或资产列表页）。
  • 导入 HttpUtil。
  • 在 aboutToAppear() 生命周期中调用GET请求，获取测试数据。
    • // Index.ets 片段
import { HttpUtil } from '../common/HttpUtil';

@Entry
@Component
struct Index {
  @State message: string = '等待加载数据...';

  aboutToAppear() {
    // 免费测试API（返回JSON格式的示例数据）
    const testUrl = 'https://jsonplaceholder.typicode.com/posts/1';
    
    HttpUtil.get(testUrl)
      .then(result => {
        console.info('网络请求成功：' + result);
        this.message = '获取成功！数据长度：' + result.length + ' 字符';
      })
      .catch(err => {
        console.error('网络请求失败：' + err);
        this.message = '请求失败，请检查网络';
      });
  }

  build() {
    Column() {
      Text(this.message)
        .fontSize(20)
        .margin(20)
      Button('重新请求')
        .onClick(() => {
          this.message = '重新请求中...';
          this.aboutToAppear();
        })
        .margin(20)
    }
    .width('100%')
    .height('100%')
  }
}

    🔍 运行测试： 点击DevEco Studio的运行按钮，观察模拟器或真机上按钮的文字变化，同时查看Log窗口的打印信息。

interface Asset {
  id: number;
  name: string;
  price: number;

@State assetList: Asset[] = [];

aboutToAppear() {
  const apiUrl = 'http://10.0.2.2:8080/assets';  // 模拟器访问宿主机
