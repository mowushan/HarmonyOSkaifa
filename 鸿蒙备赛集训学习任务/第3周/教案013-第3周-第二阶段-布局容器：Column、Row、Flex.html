🧩 知识讲解 · 布局三兄弟的故事

1. Column —— 垂直“叠叠乐”

📱 趣味比喻： Column就像自动售货机里的商品，一个按钮一个按钮从上往下排列。你放进去几个组件，它就给你竖着排几个。

核心属性：

  • space：子组件之间的间距（空隙大小）
  • alignItems：水平对齐方式（Start/Center/End）
  • justifyContent：垂直方向上的分布方式（Start/Center/End/SpaceBetween等）
// Column 基础示例
Column({ space: 10 }) {
    Text('用户名：')
        .fontSize(16)
    TextInput({ placeholder: '请输入用户名' })
        .width('90%')
    Text('密码：')
        .fontSize(16)
    TextInput({ placeholder: '请输入密码' })
        .width('90%')
    Button('登录')
        .width('90%')
}
.width('100%')
.alignItems(HorizontalAlign.Center)

2. Row —— 水平“排排坐”

🎮 趣味比喻： Row就像电影院的一排座位，大家并排坐在一起。适合做底部导航栏、按钮组。

核心属性（和Column类似，但方向不同）：

  • space：子组件之间的间距
  • alignItems：垂直对齐方式（Top/Center/Bottom）
  • justifyContent：水平方向上的分布方式
// Row 基础示例 - 底部按钮栏
Row({ space: 20 }) {
    Button('取消')
        .backgroundColor('#999')
    Button('确认')
        .backgroundColor('#0078d4')
}
.width('100%')
.justifyContent(FlexAlign.Center)

3. Flex —— 万能“变形金刚”

🦸 趣味比喻： Flex是大哥，可以随意切换方向（direction），Column和Row都是它的“马甲”。它还支持换行、缩放等高级功能。

常用属性：

  • direction：方向（FlexDirection.Column / FlexDirection.Row）
  • wrap：是否换行（FlexWrap.Wrap / NoWrap）
  • justifyContent、alignItems：对齐方式
// Flex 示例 - 可换行的标签云
Flex({ direction: FlexDirection.Row, wrap: FlexWrap.Wrap }) {
    ForEach(['鸿蒙', 'ArkTS', '布局', '组件', '开发', '大赛'], (tag: string) => {
        Text(tag)
            .padding(8)
            .backgroundColor('#e3f2fd')
            .borderRadius(16)
            .margin(5)
    })
}

⚙️ 技能实训 · 动手搭建资产录入界面

✅ 本次实训目标：用Column、Row、Flex完成“资产录入”页面的布局骨架

📁 实训准备：创建项目

  • 打开DevEco Studio，创建一个新的Empty Ability项目，命名为 AssetManager。
  • 找到 entry/src/main/ets/pages/Index.ets，这是我们今天的主战场。
  • 删除默认代码，我们将从零开始构建布局。

    • 🧪 实训任务一：Column垂直布局——搭建表单框架（15分钟）

  • 步骤1： 在最外层包裹一个Column，设置水平居中 alignItems(HorizontalAlign.Center)。
  • 步骤2： 依次添加标题、资产名称输入框、资产价值输入框、备注输入框。
  • 步骤3： 设置 space(15) 让组件之间保持舒适间距。
  • 步骤4： 运行到模拟器或真机，观察垂直排列效果。
    • // 实训一参考代码
@Entry
@Component
struct Index {
    build() {
        Column({ space: 15 }) {
            Text('资产录入')
                .fontSize(24)
                .fontWeight(FontWeight.Bold)
                .margin({ top: 20 })
            
            Text('资产名称')
                .fontSize(16)
                .alignSelf(ItemAlign.Start)
                .margin({ left: 20 })
            TextInput({ placeholder: '请输入资产名称' })
                .width('90%')
            
            Text('资产价值')
                .fontSize(16)
                .alignSelf(ItemAlign.Start)
                .margin({ left: 20 })
            TextInput({ placeholder: '请输入价值（元）' })
                .width('90%')
                .type(InputType.Number)
            
            Text('备注信息')
                .fontSize(16)
                .alignSelf(ItemAlign.Start)
                .margin({ left: 20 })
            TextArea({ placeholder: '选填' })
                .width('90%')
                .height(80)
        }
        .width('100%')
        .height('100%')
        .alignItems(HorizontalAlign.Center)
    }
}

    ✅ 检查点：运行后，所有组件应该垂直排列，输入框居中显示，标题在顶部。

    🧪 实训任务二：Row水平布局——添加按钮组（15分钟）

  • 在Column内部，TextArea的下方，插入一个Row容器。
  • Row里放置两个按钮：“保存资产”和“重置表单”。
  • 设置Row的 space(20) 和 justifyContent(FlexAlign.Center) 让按钮居中并留有空隙。
  • 给按钮添加点击事件（暂时只弹出一个提示框）。
    • // 在Column内部添加按钮组Row
Row({ space: 20 }) {
    Button('保存资产')
        .backgroundColor('#0078d4')
        .onClick(() => {
            console.info('保存按钮被点击')
            // 暂时只弹窗提示
            AlertDialog.show({ message: '功能开发中，敬请期待！' })
        })
    Button('重置表单')
        .backgroundColor('#999')
        .onClick(() => {
            console.info('重置表单')
        })
}
.justifyContent(FlexAlign.Center)
.margin({ top: 20, bottom: 30 })

    🧪 实训任务三：Flex弹性布局——添加标签推荐区（15分钟）

  • 在资产名称和输入框之间，插入一个Flex容器，显示“推荐资产类型”标签。
  • 使用 direction: FlexDirection.Row 和 wrap: FlexWrap.Wrap 实现换行效果。
  • 为每个标签添加点击事件，点击后自动填充资产名称输入框。
    • // 添加标签推荐区（放在资产名称Text之前）
Flex({ direction: FlexDirection.Row, wrap: FlexWrap.Wrap }) {
    ForEach(['笔记本电脑', '台式电脑', '投影仪', '打印机', '服务器', '交换机'], (tag: string) => {
        Text(tag)
            .padding({ left: 12, right: 12, top: 6, bottom: 6 })
            .backgroundColor('#e8f0fe')
            .borderRadius(20)
            .margin(5)
            .fontSize(14)
            .onClick(() => {
                // 这里需要绑定输入框，暂时用console
                console.info(`点击了标签：${tag}`)
                // 扩展提示：后续课程会学习状态管理，这里先埋下伏笔
                AlertDialog.show({ message: `点击了「${tag}」，后续将自动填入资产名称` })
            })
    })
}
.width('90%')
.margin({ top: 5, bottom: 5 })
