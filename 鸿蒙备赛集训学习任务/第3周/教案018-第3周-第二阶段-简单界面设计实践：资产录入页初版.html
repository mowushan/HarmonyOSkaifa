🧩 知识讲解 · 鸿蒙界面的积木块

1. TextInput —— 让用户输入信息

✍️ TextInput就像表格里的填空格，用户可以打字输入内容。

TextInput({ placeholder: '请输入资产名称' })
  .onChange((value: string) => {
    // 当输入框内容变化时，value就是用户输入的文字
  })

常用属性： placeholder(提示文字)、type(输入类型，如InputType.Number)、maxLength(最大长度)。

2. @State —— 让数据“活”起来

@State assetName: string = '';   // 资产名称
@State quantity: number = 0;      // 数量
@State purchaseDate: string = ''; // 购买日期

@State装饰的变量一旦改变，界面上用到这个变量的地方会自动刷新！这就是声明式UI的魅力。

3. Button与弹窗 —— 交互反馈

Button('提交资产')
  .onClick(() => {
    AlertDialog.show({
      message: `资产：${this.assetName}，数量：${this.quantity}`,
      confirm: { value: '确定', action: () => {} }
    });
  })

点击按钮，弹出一个提示框，展示我们刚才填的信息，成就感满满！

⚙️ 技能实训 · 亲手打造资产录入页

✅ 本次实训目标：完成一个完整可交互的资产录入页面

📁 步骤0：创建项目与页面

  • 打开DevEco Studio，创建新项目（Empty Ability），项目名：AssetManagement，保存位置自己定。
  • 在 entry/src/main/ets/pages 目录下，右键新建 AssetEntryPage.ets。
  • 在 Index.ets 中，用路由跳转到新页面（老师先演示，大家先直接设置Index页面为AssetEntryPage内容也行，重点练习界面）。
  • 为了快速聚焦，我们直接在 Index.ets 中编写本次课的界面，替换原有代码。

    • 🔧 实训一：搭建基础结构（15分钟）

  • 打开 Index.ets，清空原有内容，编写最外层结构：
    • @Entry
@Component
struct AssetEntryPage {
  build() {
    Column() {
      Text('资产录入')
        .fontSize(28)
        .fontWeight(FontWeight.Bold)
        .margin({ top: 20, bottom: 20 })
      
      // 我们稍后在这里添加表单组件
    }
    .width('100%')
    .height('100%')
    .padding(20)
    .backgroundColor('#F5F5F5')
  }
}

    ✅ 运行预览器，你应该看到顶部大字“资产录入”和灰色背景。如果报错，检查括号是否匹配，@Component装饰器是否正确。

    🔧 实训二：添加资产名称输入框（15分钟）

  • 在Text下方添加状态变量：
    • @State assetName: string = '';
  • 添加TextInput组件，并绑定@State：
    • TextInput({ placeholder: '请输入资产名称' })
  .width('100%')
  .margin({ bottom: 15 })
  .onChange((value: string) => {
    this.assetName = value;
  })
  • 在下方加一个Text组件，实时显示输入内容（用于调试）：
    • Text(`当前输入：${this.assetName}`)
  .fontSize(14)
  .fontColor('#666')
  .margin({ bottom: 10 })

    🎉 验证： 在输入框打字，下方的文字会同步变化！这证明@State绑定成功。

    🔧 实训三：添加数量与日期字段（15分钟）

  • 再添加两个状态变量：
    • @State quantity: number = 0;
@State purchaseDate: string = '';
  • 添加数量输入框（限制只能输入数字）：
    • TextInput({ placeholder: '请输入数量' })
  .width('100%')
  .margin({ bottom: 15 })
  .type(InputType.Number)
  .onChange((value: string) => {
    this.quantity = parseInt(value) || 0;
  })
  • 添加日期输入框（简单起见用普通文本，后续再教日期选择器）：
    • TextInput({ placeholder: '购买日期 (例如 2026-03-22)' })
  .width('100%')
  .margin({ bottom: 20 })
  .onChange((value: string) => {
    this.purchaseDate = value;
  })

    📸 此刻你的界面应该有三个输入框：资产名称、数量、购买日期，并且每个输入都能实时绑定数据。

    🔧 实训四：添加提交按钮与弹窗（15分钟）

  • 在Column末尾添加按钮：
    • Button('提交资产')
  .width('100%')
  .height(48)
  .backgroundColor('#0078D4')
  .fontColor(Color.White)
  .borderRadius(24)
  .onClick(() => {
    AlertDialog.show({
      title: '资产信息',
      message: `名称：${this.assetName}\n数量：${this.quantity}\n日期：${this.purchaseDate}`,
      confirm: {
        value: '确定',
        action: () => {
          console.info('用户确认提交');
        }
      }
    });
  })

    💡 点击按钮，弹窗显示填写的资产信息！如果message显示空白，说明绑定有问题，检查onChange是否正确更新了@State。

    🔧 实训五：布局美化 & 间距调整（10分钟）

  • 给每个输入框之间添加更多margin，让界面呼吸感更强。
  • 给整个Column添加.alignItems(HorizontalAlign.Center)，让内容水平居中？不，表单通常左对齐，我们保持默认。
  • 给每个输入框添加圆角和背景色：
    • .backgroundColor('#FFFFFF')
.borderRadius(12)
.padding({ left: 12, right: 12 })
  • 还可以给按钮增加阴影效果：.shadow({ radius: 8, color: '#CCCCCC' })

    • 🔧 实训六（挑战）：预览区域卡片化（10分钟）

    在提交按钮上方，添加一个“预览卡片”，展示当前填写的资产信息摘要，样式更专业。

    Column() {
  Text('📋 预览')
    .fontSize(16)
    .fontWeight(FontWeight.Bold)
    .margin({ bottom: 8 })
  Text(`资产名称：${this.assetName || '未填写'}`)
  Text(`数量：${this.quantity || 0}`)
  Text(`购买日期：${this.purchaseDate || '未填写'}`)
}
.width('100%')
.backgroundColor('#E8F0FE')
.borderRadius(16)
.padding(12)
.margin({ top: 10, bottom: 15 })

    ✅ 添加后，用户填写时右侧/下方实时显示预览，体验感拉满！
