🧩 知识讲解 · 核心功能集成思路

1. 整体架构回顾 · 三大模块协同工作

🏗️ 我们的资产管理系统包含三大模块：

  • 📱 UI模块（界面层）：资产录入页、资产列表页、资产详情/编辑页
  • 💾 数据模块（数据层）：数据库帮助类（增删改查）、Preferences存储（设置）
  • 🌐 网络模块（服务层）：HTTP请求封装、JSON解析、云端数据同步

数据流向图：

录入页(输入) → 数据验证 → 调用数据库帮助类.save() → 存入本地SQLite
列表页启动 → 调用数据库帮助类.queryAll() → 获取资产列表 → UI展示
详情页编辑 → 调用数据库帮助类.update() → 更新本地数据 → 可选同步到云端
云端获取 → 调用网络模块.fetchRemote() → JSON解析 → 合并本地数据

2. 模块集成关键点 · 页面间数据传递

📤 路由传参

列表页点击某条资产，需要把资产ID传给详情页，详情页根据ID查询数据库展示。

🔄 页面刷新

录入新资产后，返回列表页需要自动刷新列表。使用 @State + 事件总线 或 router.back() 携带刷新标识。

🔁 数据同步

本地存储为主，云端为辅。增删改操作同时更新本地和云端（异步）。

    • 🧪 实训任务一：完善数据库帮助类（15分钟）

    DatabaseHelper.ets 核心代码（教师投影展示，学生逐行理解）：

    import relationalStore from '@ohos.data.relationalStore';
import { AssetModel } from './AssetModel';

const SQL_CREATE_TABLE = `
    CREATE TABLE IF NOT EXISTS assets (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT NOT NULL,
        category TEXT,
        quantity INTEGER DEFAULT 1,
        price REAL,
        purchaseDate TEXT,
        remark TEXT
    )
`;

export class DatabaseHelper {
    private static instance: DatabaseHelper;
    private rdbStore: relationalStore.RdbStore | null = null;

    static getInstance(): DatabaseHelper {
        if (!DatabaseHelper.instance) {
            DatabaseHelper.instance = new DatabaseHelper();
        }
        return DatabaseHelper.instance;
    }

    async init(context: Context): Promise<void> {
        const config = {
            name: 'asset.db',
            securityLevel: relationalStore.SecurityLevel.S1
        };
        this.rdbStore = await relationalStore.getRdbStore(context, config);
        await this.rdbStore.executeSql(SQL_CREATE_TABLE);
        console.info('数据库初始化成功');
    }

    // 插入资产
    async insertAsset(asset: AssetModel): Promise<number> {
        if (!this.rdbStore) throw new Error('数据库未初始化');
        const valueBucket = {
            name: asset.name,
            category: asset.category,
            quantity: asset.quantity,
            price: asset.price,
            purchaseDate: asset.purchaseDate,
            remark: asset.remark
        };
        return await this.rdbStore.insert('assets', valueBucket);
    }

    // 查询所有资产
    async queryAllAssets(): Promise<AssetModel[]> {
        if (!this.rdbStore) return [];
        const predicates = new relationalStore.RdbPredicates('assets');
        const resultSet = await this.rdbStore.query(predicates);
        const assets: AssetModel[] = [];
        while (resultSet.goToNextRow()) {
            assets.push({
                id: resultSet.getLong(resultSet.getColumnIndex('id')),
                name: resultSet.getString(resultSet.getColumnIndex('name')),
                category: resultSet.getString(resultSet.getColumnIndex('category')),
                quantity: resultSet.getLong(resultSet.getColumnIndex('quantity')),
                price: resultSet.getDouble(resultSet.getColumnIndex('price')),
                purchaseDate: resultSet.getString(resultSet.getColumnIndex('purchaseDate')),
                remark: resultSet.getString(resultSet.getColumnIndex('remark'))
            });
        }
        resultSet.close();
        return assets;
    }

    // 更新资产
    async updateAsset(asset: AssetModel): Promise<number> {
        if (!this.rdbStore) throw new Error('数据库未初始化');
        const valueBucket = {
            name: asset.name,
            category: asset.category,
            quantity: asset.quantity,
            price: asset.price,
            purchaseDate: asset.purchaseDate,
            remark: asset.remark
        };
        const predicates = new relationalStore.RdbPredicates('assets');
        predicates.equalTo('id', asset.id);
        return await this.rdbStore.update(valueBucket, predicates);
    }

    // 删除资产
    async deleteAsset(id: number): Promise<number> {
        if (!this.rdbStore) throw new Error('数据库未初始化');
        const predicates = new relationalStore.RdbPredicates('assets');
        predicates.equalTo('id', id);
        return await this.rdbStore.delete(predicates);
    }
}

    🧪 实训任务二：资产列表页集成数据库（20分钟）

    • import router from '@ohos.router';
import { DatabaseHelper } from '../common/DatabaseHelper';
import { AssetModel } from '../common/AssetModel';

@Entry
@Component
struct AssetListPage {
    @State assets: AssetModel[] = [];

    aboutToAppear() {
        this.loadData();
    }

    async loadData() {
        try {
            this.assets = await DatabaseHelper.getInstance().queryAllAssets();
        } catch (err) {
            console.error('加载数据失败：' + err);
        }
    }

    onPageShow() {
        // 从其他页面返回时刷新列表（录入/编辑后）
        this.loadData();
    }

    build() {
        Column() {
            Row() {
                Text('资产管理')
                    .fontSize(24)
                    .fontWeight(FontWeight.Bold)
                Blank()
                Button('+ 添加资产')
                    .onClick(() => {
                        router.pushUrl({ url: 'pages/AssetAddPage' });
                    })
            }
            .width('100%')
            .padding(12)

            List() {
                ForEach(this.assets, (asset: AssetModel) => {
                    ListItem() {
                        Row() {
                            Column() {
                                Text(asset.name)
                                    .fontSize(18)
                                    .fontWeight(FontWeight.Medium)
                                Text(`数量：${asset.quantity}  价格：¥${asset.price}`)
                                    .fontSize(14)
                                    .fontColor('#666')
                            }
                            .alignItems(HorizontalAlign.Start)
                            .layoutWeight(1)
                            Text('详情')
                                .fontColor('#0078d4')
                                .onClick(() => {
                                    router.pushUrl({
                                        url: 'pages/AssetDetailPage',
                                        params: { id: asset.id }
                                    });
                                })
                        }
                        .width('100%')
                        .padding(12)
                        .justifyContent(FlexAlign.SpaceBetween)
                    }
                })
            }
            .layoutWeight(1)
        }
        .width('100%')
        .height('100%')
    }
}

    🧪 实训任务三：资产录入页集成存储（15分钟）

  • 创建 AssetAddPage.ets，包含表单输入框（名称、分类、数量、价格等）。
  • 使用 @State 绑定输入值，如 @State name: string = ''。
  • 实现保存按钮逻辑：校验输入 → 构造 AssetModel → 调用 insertAsset → 返回上一页。
  • 返回时携带刷新标识：router.back() 会自动触发列表页 onPageShow。
    • import router from '@ohos.router';
import { DatabaseHelper } from '../common/DatabaseHelper';
import { AssetModel } from '../common/AssetModel';

@Entry
@Component
struct AssetAddPage {
    @State name: string = '';
    @State category: string = '';
    @State quantity: string = '';
    @State price: string = '';
    @State purchaseDate: string = '';
    @State remark: string = '';

    async saveAsset() {
        if (!this.name.trim()) {
            alert('请填写资产名称');
            return;
        }
        const asset: AssetModel = {
            name: this.name,
            category: this.category,
            quantity: parseInt(this.quantity) || 0,
            price: parseFloat(this.price) || 0,
            purchaseDate: this.purchaseDate,
            remark: this.remark
        };
        try {
            await DatabaseHelper.getInstance().insertAsset(asset);
            router.back();  // 保存成功，返回列表页
        } catch (err) {
            alert('保存失败：' + err);
        }
    }

    build() {
        Column() {
            Text('添加资产')
                .fontSize(24)
                .fontWeight(FontWeight.Bold)
                .margin({ bottom: 20 })

            TextInput({ placeholder: '资产名称', text: this.name })
                .onChange((val) => this.name = val)
            TextInput({ placeholder: '分类', text: this.category })
                .onChange((val) => this.category = val)
            TextInput({ placeholder: '数量', text: this.quantity })
                .type(InputType.Number)
                .onChange((val) => this.quantity = val)
            TextInput({ placeholder: '单价', text: this.price })
                .type(InputType.Number)
                .onChange((val) => this.price = val)
            TextInput({ placeholder: '购买日期', text: this.purchaseDate })
                .onChange((val) => this.purchaseDate = val)
            TextArea({ placeholder: '备注', text: this.remark })
                .onChange((val) => this.remark = val)

            Button('保存资产')
                .width('100%')
                .margin({ top: 20 })
                .onClick(() => this.saveAsset())
        }
        .padding(20)
        .width('100%')
        .height('100%')
    }
}

    🧪 实训任务四：资产详情/编辑页（10分钟）

  • 创建 AssetDetailPage.ets，从路由参数获取 assetId。
  • 在 aboutToAppear 中根据 ID 查询详情，并填充到 @State 变量。
  • 实现“编辑”模式：点击编辑按钮启用输入框，保存时调用 updateAsset。
  • 添加“删除”按钮，调用 deleteAsset 后返回列表页。
    • import router from '@ohos.router';
import { DatabaseHelper } from '../common/DatabaseHelper';
import { AssetModel } from '../common/AssetModel';

@Entry
@Component
struct AssetDetailPage {
    @State asset: AssetModel | null = null;
    @State isEdit: boolean = false;
    private assetId: number = -1;

    aboutToAppear() {
        const params = router.getParams() as Record<string, number>;
        this.assetId = params['id'];
        this.loadDetail();
    }

    async loadDetail() {
        const all = await DatabaseHelper.getInstance().queryAllAssets();
        this.asset = all.find(item => item.id === this.assetId) || null;
    }

    async updateAsset() {
        if (!this.asset) return;
        await DatabaseHelper.getInstance().updateAsset(this.asset);
        this.isEdit = false;
        alert('更新成功');
    }

    async deleteAsset() {
        if (!this.asset) return;
        await DatabaseHelper.getInstance().deleteAsset(this.asset.id!);
        router.back();
    }

    build() {
        if (!this.asset) {
            return Column() { Text('加载中...') }.width('100%').height('100%');
        }
        return Column() {
            Text(isEdit ? '编辑资产' : '资产详情')
                .fontSize(24)
                .fontWeight(FontWeight.Bold)

            if (isEdit) {
                TextInput({ text: this.asset.name })
                    .onChange(val => this.asset!.name = val)
                TextInput({ text: this.asset.category })
                    .onChange(val => this.asset!.category = val)
                // ... 其他字段类似
                Button('保存修改')
                    .onClick(() => this.updateAsset())
                Button('取消')
                    .onClick(() => this.isEdit = false)
            } else {
                Text(`名称：${this.asset.name}`)
                Text(`分类：${this.asset.category}`)
                Text(`数量：${this.asset.quantity}`)
                Text(`单价：${this.asset.price}`)
                Row() {
                    Button('编辑')
                        .onClick(() => this.isEdit = true)
                    Button('删除')
                        .backgroundColor('#d32f2f')
                        .onClick(() => this.deleteAsset())
                }
            }
        }
        .padding(20)
    }
}
