🧩 知识讲解 · 编辑与删除的核心原理

1. 编辑功能 · 三步走

✏️ 编辑的本质： 把选中的数据显示到编辑表单 → 用户修改 → 保存更新到数据库。

实现步骤：

  1. 列表页跳转： 点击“编辑”按钮，携带资产对象跳转到编辑页。
  2. 编辑页接收数据： 通过路由参数获取要编辑的资产信息，填充到输入框。
  3. 保存修改： 用户点击保存，调用数据库更新方法，更新成功后返回列表页并刷新。
// 列表页跳转到编辑页（携带参数）
router.pushUrl({
    url: 'pages/AssetEdit',
    params: { asset: currentAsset }
})

// 编辑页接收参数
import router from '@ohos.router';
onPageShow() {
    const params = router.getParams();
    if (params && params.asset) {
        this.asset = params.asset;
        this.name = this.asset.name;
        this.price = this.asset.price.toString();
    }
}

2. 删除功能 · 安全第一

🗑️ 删除原则： 必须二次确认！使用系统弹窗让用户再次确认是否删除。

实现要点：

  • 在列表项上添加“删除”按钮。
  • 点击时弹出 AlertDialog 确认框。
  • 用户确认后，调用数据库删除方法。
  • 删除成功后刷新列表数据。

3. 数据同步刷新技巧

🔄 编辑或删除后，返回列表页时如何自动刷新？

两种常用方法：

  • 方法一： 在列表页的 onPageShow() 生命周期中重新加载数据。
  • 方法二： 使用事件总线（EventHub）通知列表页刷新。

今天我们先使用方法一，简单又可靠！

// 列表页：每次页面显示时重新加载数据
onPageShow() {
    this.loadAssetList();  // 重新查询数据库
}

🔧 实训一：创建编辑页面（15分钟）

  • 新建编辑页面文件： 在 pages 目录下新建 AssetEdit.ets。
  • 设计编辑界面： 包含名称输入框、价格输入框、保存按钮和返回按钮。
  • 接收路由参数： 在页面中获取传递过来的资产对象，并将数据绑定到输入框。
  • 编写保存逻辑： 调用数据库更新方法，更新成功后返回上一页。
    • // AssetEdit.ets 完整代码示例
import router from '@ohos.router';
import { AssetDatabase } from '../database/AssetDatabase';

@Entry
@Component
struct AssetEdit {
    @State assetId: number = 0;
    @State name: string = '';
    @State price: string = '';
    private db: AssetDatabase = new AssetDatabase();

    aboutToAppear() {
        const params = router.getParams();
        if (params && params['asset']) {
            const asset = params['asset'] as Record<string, Object>;
            this.assetId = asset['id'] as number;
            this.name = asset['name'] as string;
            this.price = String(asset['price']);
        }
    }

    async saveAsset() {
        if (!this.name.trim()) {
            promptAction.showToast({ message: '请输入资产名称' });
            return;
        }
        const priceNum = parseFloat(this.price);
        if (isNaN(priceNum)) {
            promptAction.showToast({ message: '请输入正确的价格' });
            return;
        }
        await this.db.updateAsset(this.assetId, this.name, priceNum);
        router.back();
    }

    build() {
        Column() {
            Text('编辑资产')
                .fontSize(24)
                .fontWeight(FontWeight.Bold)
                .margin(20)
            TextInput({ placeholder: '资产名称', text: this.name })
                .onChange((value) => { this.name = value; })
                .margin(10)
            TextInput({ placeholder: '资产价格', text: this.price })
                .type(InputType.Number)
                .onChange((value) => { this.price = value; })
                .margin(10)
            Button('保存修改')
                .width('90%')
                .onClick(() => this.saveAsset())
                .margin(10)
            Button('返回')
                .width('90%')
                .backgroundColor(Color.Gray)
                .onClick(() => router.back())
                .margin(10)
        }
        .width('100%')
        .height('100%')
        .justifyContent(FlexAlign.Center)
    }
}

async updateAsset(id: number, name: string, price: number): Promise<boolean> {
    let result = await this.store?.update(
        "UPDATE asset SET name = ?, price = ? WHERE id = ?",
        [name, price, id]
    );
    return result !== undefined && result.changes > 0;
}

async deleteAsset(id: number): Promise<boolean> {
    let result = await this.store?.delete("DELETE FROM asset WHERE id = ?", [id]);
    return result !== undefined && result.changes > 0;
}
