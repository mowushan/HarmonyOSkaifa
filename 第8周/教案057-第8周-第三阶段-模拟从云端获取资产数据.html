🧩 知识讲解 · 网络请求与云端数据

1. 什么是网络请求？—— 像点外卖一样简单

📱 趣味类比： 你在手机上打开外卖App，点击“美食”按钮 → App向服务器发送请求（我要看美食列表） → 服务器把美食数据打包发回来 → App展示给你看。这就是网络请求！

两个常用方法：

  • 🔹 GET：向服务器“要”数据（比如获取资产列表）
  • 🔹 POST：向服务器“提交”数据（比如新增资产）

本节课我们先用GET模拟从云端获取资产数据！

2. 鸿蒙中的网络请求 · http模块

import http from '@ohos.net.http';

// 创建HTTP请求对象
let httpRequest = http.createHttp();

// 发起GET请求
httpRequest.request('https://api.example.com/assets', {
    method: http.RequestMethod.GET,
    header: { 'Content-Type': 'application/json' }
}, (err, data) => {
    if (err) {
        console.error('请求失败: ' + JSON.stringify(err));
        return;
    }
    console.info('请求成功: ' + data.result);
});

📌 知识点拆解：

  • createHttp()：创建一个“电话机”（HTTP客户端）
  • request()：拨打电话，发送请求
  • method：告诉服务器你要GET还是POST
  • 回调函数：服务器“接电话”后的处理逻辑

3. JSON数据格式 —— 服务器的“通用语言”

📦 JSON长什么样？

{
  "code": 200,
  "data": [
    {"id": 1, "name": "笔记本电脑", "price": 5999},
    {"id": 2, "name": "显示器", "price": 1299}
  ]
}

🔧 如何解析？

let jsonObj = JSON.parse(jsonString);
let assetList = jsonObj.data;
console.log(assetList[0].name); // 输出：笔记本电脑

⚠️ 重要提醒： 网络请求是异步操作，就像“发微信后等回复”，代码不会停在原地等待，需要用回调或async/await处理。

    🧪 实训任务三：编写网络请求代码（30分钟）

  • 在 CloudAssetsPage.ets 中编写页面结构。
  • 导入http模块：import http from '@ohos.net.http';
  • 编写请求函数，获取数据并解析。
    • import http from '@ohos.net.http';
import router from '@ohos.router';

@Entry
@Component
struct CloudAssetsPage {
  @State assetList: Array<AssetItem> = [];
  @State isLoading: boolean = false;

  aboutToAppear() {
    this.fetchAssetsFromCloud();
  }

  async fetchAssetsFromCloud() {
    this.isLoading = true;
    let httpRequest = http.createHttp();
    
    try {
      let response = await httpRequest.request(
        'https://jsonplaceholder.typicode.com/posts',
        {
          method: http.RequestMethod.GET,
          connectTimeout: 10000,
          readTimeout: 10000,
          header: { 'Content-Type': 'application/json' }
        }
      );
      
      if (response.responseCode === 200) {
        // 解析JSON数据
        let rawData = JSON.parse(response.result);
        // 将假数据转换为资产格式
        let assets: AssetItem[] = rawData.map((item: any) => {
          return {
            id: item.id,
            name: `资产${item.id}`,
            description: item.title,
            price: Math.floor(Math.random() * 10000)
          };
        });
        this.assetList = assets;
        console.info(`成功获取${this.assetList.length}条资产数据`);
      } else {
        console.error(`请求失败，状态码：${response.responseCode}`);
      }
    } catch (error) {
      console.error('网络请求异常：' + JSON.stringify(error));
    } finally {
      this.isLoading = false;
      httpRequest.destroy();
    }
  }

  build() {
    Column() {
      // 标题栏
      Row() {
        Text('云端资产列表')
          .fontSize(24)
          .fontWeight(FontWeight.Bold)
          .margin({ left: 20 })
        Blank()
        Button('返回')
          .onClick(() => {
            router.back();
          })
          .margin({ right: 20 })
      }
      .width('100%')
      .height(60)
      .backgroundColor('#F1F3F5')

      // 加载中提示
      if (this.isLoading) {
        LoadingProgress()
          .width(50)
          .height(50)
          .margin({ top: 50 })
      } else {
        // 资产列表
        List() {
          ForEach(this.assetList, (asset: AssetItem, index: number) => {
            ListItem() {
              Column() {
                Text(`【${asset.name}】`)
                  .fontSize(18)
                  .fontWeight(FontWeight.Medium)
                Text(`描述：${asset.description}`)
                  .fontSize(14)
                  .fontColor('#666')
                Text(`价格：¥${asset.price}`)
                  .fontSize(14)
                  .fontColor('#E67E22')
              }
              .width('100%')
              .padding(12)
              .alignItems(HorizontalAlign.Start)
            }
            .margin({ top: 8, left: 16, right: 16 })
            .backgroundColor('#FFFFFF')
            .borderRadius(12)
          }, (asset: AssetItem) => asset.id.toString())
        }
        .width('100%')
        .layoutWeight(1)
      }
    }
    .width('100%')
    .height('100%')
    .backgroundColor('#F8F9FA')
  }
}

// 资产数据模型
interface AssetItem {
  id: number;
  name: string;
  description: string;
  price: number;
}

    📌 代码解读：

    • async/await：让异步代码像同步一样写，更易读。
    • try/catch/finally：捕获网络错误，保证程序不崩溃。
    • response.responseCode === 200：HTTP 200表示请求成功。
    • JSON.parse()：把服务器返回的文本转成JavaScript对象。
    • httpRequest.destroy()：用完销毁，释放资源。
