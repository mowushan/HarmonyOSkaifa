🧩 知识讲解 · 栅格布局（Grid）从入门到上手

1. 什么是栅格布局？—— 像搭积木一样排组件

🎲 趣味引入： 想象一下，你有一个4×3的乐高底板（4列3行），每个格子可以放一块乐高。Grid布局就是帮你把界面划分成这样的“隐形格子”，然后把组件按格子摆放。

Grid vs Flex：

  • 📏 Flex（弹性布局）：一维布局，要么水平排，要么垂直排，适合导航栏、工具栏。
  • 📐 Grid（栅格布局）：二维布局，同时控制行和列，适合相册、仪表盘、商品展示墙。

2. Grid核心属性 —— 学会这三招，Grid就通了

📌 columnsTemplate

定义列的数量和宽度。例如：'1fr 1fr 1fr' 表示三等分列；'100px 1fr 2fr' 表示第一列固定100px，后面按比例分配。

📌 rowsTemplate

定义行的高度。例如：'auto auto' 表示两行高度自适应内容；'80px 1fr' 第一行80px，第二行占满剩余空间。

📌 columnsGap / rowsGap

设置网格之间的间距。例如：columnsGap: 12, rowsGap: 12 让网格之间留出呼吸空间。

// 最简单的Grid示例：3列等宽，2行自动高度，间距12 Grid() { GridItem() { Text('1') } GridItem() { Text('2') } GridItem() { Text('3') } GridItem() { Text('4') } GridItem() { Text('5') } GridItem() { Text('6') } } .columnsTemplate('1fr 1fr 1fr') .rowsTemplate('auto auto') .columnsGap(12) .rowsGap(12)

3. fr单位是什么？—— 比例分配剩余空间

🍕 生活类比： 假如你有一张披萨，要分给三个人，比例是2:1:1，那么第一个人拿一半，后两个人各拿四分之一。fr单位就是这种“比例分配”！

'2fr 1fr 1fr' = 把剩余宽度分成4份，第一列占2份，第二三列各占1份。

4. 跨行跨列 —— 让网格项更灵活

GridItem() { Text('大标题') } .columnStart(1) // 从第1列开始 .columnEnd(3) // 到第3列结束（即占2列） .rowStart(1) .rowEnd(2)

💡 记忆口诀： columnStart/End 控制左右跨越，rowStart/End 控制上下跨越。

⚙️ 技能实训 · 在VSCode中玩转栅格布局

✅ 实训目标：从零实现一个“资产统计看板”，熟练掌握Grid布局

🔧 实训准备：创建项目文件

  • 打开 DevEco Studio，在之前的鸿蒙项目中，新建一个页面：AssetDashboard.ets（右键 entry/src/main/ets/pages → New → ArkTS File）。
  • 在项目结构中，找到 pages/index.ets，暂时不用改，我们新建页面独立测试。
  • 在 AssetDashboard.ets 中编写基础结构：@Entry @Component struct AssetDashboard { build() { Column() { ... } } }。

    • 🧪 实训任务一：第一个Grid —— 3列2行网格（15分钟）

  • 在 build() 的 Column 中，添加一个 Grid 容器。
  • 设置 .columnsTemplate('1fr 1fr 1fr')，三列等宽。
  • 设置 .rowsTemplate('auto auto')，两行高度自适应。
  • 添加6个 GridItem，每个 GridItem 里放一个带背景色的 Text，内容分别为“资产1”到“资产6”。
  • 运行模拟器，查看效果。
    • // 实训一参考代码 Grid() { GridItem() { Text('资产1').width('100%').height(80).backgroundColor('#4CAF50').textAlign(TextAlign.Center) } GridItem() { Text('资产2').width('100%').height(80).backgroundColor('#2196F3').textAlign(TextAlign.Center) } GridItem() { Text('资产3').width('100%').height(80).backgroundColor('#FF9800').textAlign(TextAlign.Center) } GridItem() { Text('资产4').width('100%').height(80).backgroundColor('#9C27B0').textAlign(TextAlign.Center) } GridItem() { Text('资产5').width('100%').height(80).backgroundColor('#E91E63').textAlign(TextAlign.Center) } GridItem() { Text('资产6').width('100%').height(80).backgroundColor('#00BCD4').textAlign(TextAlign.Center) } } .columnsTemplate('1fr 1fr 1fr') .rowsTemplate('auto auto') .columnsGap(12) .rowsGap(12) .width('100%') .padding(16)

    🎉 运行成功！你会看到一个3列2行的彩色卡片网格。

    🧪 实训任务二：比例列宽 + 跨列效果（20分钟）

  • 修改 .columnsTemplate('2fr 1fr 1fr')，让第一列宽度是后两列的两倍。
  • 将第一个 GridItem 设置为跨两列：添加 .columnStart(1).columnEnd(3)。
  • 为每个 GridItem 添加点击事件（.onClick(() => {})），弹出提示框显示当前资产名称。
  • 观察布局变化，第一列变宽，且第一个卡片占据了两列宽度。
    • // 实训二关键代码 Grid() { GridItem() { Text('总资产概览').fontSize(20).fontWeight(FontWeight.Bold) } .columnStart(1) .columnEnd(3) .backgroundColor('#FFC107') .height(100) .borderRadius(12) GridItem() { Text('设备数量: 42') } GridItem() { Text('维护中: 3') } // ... 其余 GridItem } .columnsTemplate('2fr 1fr 1fr') .rowsTemplate('auto auto auto') .columnsGap(12) .rowsGap(12)

    🧪 实训任务三：实战 —— 资产统计看板（30分钟）

    🎯 项目需求： 为资产管理系统设计一个顶部统计看板，使用 Grid 展示四个核心指标，下方再用 Grid 展示设备分类占比。

    设计稿：

    • 第一行（1×4网格）：总资产数、总价值、在用设备、待维修
    • 第二行（2×2网格）：设备类型分布（电脑、打印机、服务器、网络设备）
  • 创建第一个 Grid（统计卡片），.columnsTemplate('1fr 1fr 1fr 1fr')，每个 GridItem 内包含数值和说明。
  • 创建第二个 Grid（分类展示），.columnsTemplate('1fr 1fr')，共4个 GridItem（自动排成2行）。
  • 给每个卡片添加圆角、阴影效果，提升视觉体验。
  • 增加点击响应：点击“电脑”卡片，弹出提示“电脑类设备共23台”。
  • 将两个 Grid 放在 Column 中，中间留出间距。
