🧩 知识讲解 · Canvas绘图秘籍

1. Canvas是什么？—— 鸿蒙里的“电子画板”

🎨 趣味类比： Canvas就像一个空白的画板，我们可以用代码当画笔，在上面画线、画圆、画矩形，甚至可以画图表、游戏画面！

Canvas的核心能力：

  • 📐 绘制图形：矩形、圆形、线条、弧线、路径。
  • 🎨 设置样式：颜色、渐变、阴影、线条粗细。
  • 📊 数据可视化：柱状图、折线图、饼图。
  • 🖼️ 绘制图片：将图片画到Canvas上。

📣 在鸿蒙中，Canvas组件通过CanvasRenderingContext2D对象来操作。

2. Canvas基础API · 矩形与颜色

// 1. 获取Canvas的绘图上下文 private ctx: CanvasRenderingContext2D | null = null; // 2. 设置画布尺寸和背景 Canvas(this.ctx) .width('100%') .height(300) .onReady(() => { if (this.ctx) { // 画一个红色矩形 this.ctx.fillStyle = '#ff0000'; this.ctx.fillRect(50, 50, 100, 80); // 画一个蓝色边框的矩形 this.ctx.strokeStyle = '#0078d4'; this.ctx.lineWidth = 3; this.ctx.strokeRect(180, 50, 100, 80); } })

3. 绘制圆形与路径

// 画圆：arc(x, y, 半径, 起始弧度, 结束弧度) this.ctx.beginPath(); this.ctx.arc(150, 150, 50, 0, 2 * Math.PI); this.ctx.fillStyle = '#ffaa00'; this.ctx.fill(); this.ctx.strokeStyle = '#333'; this.ctx.stroke(); // 画一条直线路径 this.ctx.beginPath(); this.ctx.moveTo(50, 200); this.ctx.lineTo(200, 200); this.ctx.lineWidth = 4; this.ctx.stroke();

4. 绘制文字与渐变

// 绘制文字 this.ctx.font = '20px "Microsoft YaHei"'; this.ctx.fillStyle = '#004d7c'; this.ctx.fillText('鸿蒙集训队', 50, 280); // 线性渐变 const gradient = this.ctx.createLinearGradient(0, 0, 200, 0); gradient.addColorStop(0, '#ff0000'); gradient.addColorStop(1, '#0000ff'); this.ctx.fillStyle = gradient; this.ctx.fillRect(50, 300, 200, 50);

🔧 实训一：创建Canvas画板（15分钟）

  • 在build()中添加Canvas组件，设置宽高和onReady回调。
  • 在onReady中获取CanvasRenderingContext2D实例并保存到成员变量。
  • 验证： 运行预览，看到空白Canvas区域即成功。
    • // 示例代码结构 @Component export struct AssetChart { private ctx: CanvasRenderingContext2D | null = null; build() { Column() { Canvas(this.ctx) .width('100%') .height(400) .onReady(() => { console.log('Canvas准备就绪'); this.drawDemo(); }) } .padding(20) } drawDemo() { if (!this.ctx) return; // 绘制测试矩形 this.ctx.fillStyle = '#0078d4'; this.ctx.fillRect(20, 20, 100, 60); } }

    🔧 实训二：绘制资产统计柱状图（30分钟）

  • 准备数据： 定义一个数组存储各分类资产数量，例如 const data = [42, 28, 35, 51, 19];，对应“电脑、显示器、键盘、鼠标、打印机”。
  • 计算柱状图参数： 画布宽度400，每根柱子宽度40，间距20，起始x=40。
  • 循环绘制： 根据数据值计算柱子高度，设置不同颜色。
  • 添加X轴标签： 在每个柱子下方绘制分类名称。
  • 添加Y轴刻度： 绘制水平辅助线和数字。
    • drawBarChart() { if (!this.ctx) return; const data = [42, 28, 35, 51, 19]; const categories = ['电脑', '显示器', '键盘', '鼠标', '打印机']; const barWidth = 40; const gap = 20; let startX = 50; const maxValue = 60; // Y轴最大值 // 清空画布（先画白色背景） this.ctx.fillStyle = '#ffffff'; this.ctx.fillRect(0, 0, 400, 300); for (let i = 0; i < data.length; i++) { const height = (data[i] / maxValue) * 200; const y = 250 - height; // 不同柱子不同颜色 const colors = ['#5470c6', '#fac858', '#ee6666', '#73c0de', '#3ba272']; this.ctx.fillStyle = colors[i % colors.length]; this.ctx.fillRect(startX + i * (barWidth + gap), y, barWidth, height); // 绘制数值文本 this.ctx.fillStyle = '#333'; this.ctx.font = '12px sans-serif'; this.ctx.fillText(data[i].toString(), startX + i * (barWidth + gap) + 10, y - 5); // 绘制X轴标签 this.ctx.fillStyle = '#666'; this.ctx.fillText(categories[i], startX + i * (barWidth + gap) + 5, 270); } // 绘制坐标轴 this.ctx.beginPath(); this.ctx.moveTo(30, 250); this.ctx.lineTo(380, 250); // X轴 this.ctx.moveTo(30, 250); this.ctx.lineTo(30, 30); // Y轴 this.ctx.stroke(); }

    🔧 实训三：绘制资产占比饼图（25分钟）

  • 准备资产分类数据：const pieData = [{name:'电脑',value:42},{name:'显示器',value:28},{name:'键盘',value:35}];
  • 计算总值，确定每个扇区的起始角度和结束角度。
  • 使用 ctx.beginPath() 和 ctx.arc() 绘制扇区，不同颜色填充。
  • 在饼图旁边添加图例，显示每个分类的颜色和名称。
    • drawPieChart() { if (!this.ctx) return; const data = [ { name: '电脑', value: 42, color: '#5470c6' }, { name: '显示器', value: 28, color: '#fac858' }, { name: '键盘', value: 35, color: '#ee6666' }, { name: '鼠标', value: 51, color: '#73c0de' }, { name: '打印机', value: 19, color: '#3ba272' } ]; const total = data.reduce((sum, item) => sum + item.value, 0); let startAngle = -Math.PI / 2; // 从12点钟方向开始 const centerX = 150; const centerY = 150; const radius = 100; for (const item of data) { const angle = (item.value / total) * 2 * Math.PI; const endAngle = startAngle + angle; this.ctx.beginPath(); this.ctx.moveTo(centerX, centerY); this.ctx.arc(centerX, centerY, radius, startAngle, endAngle); this.ctx.closePath(); this.ctx.fillStyle = item.color; this.ctx.fill(); startAngle = endAngle; } // 绘制图例 let legendY = 50; for (let i = 0; i < data.length; i++) { this.ctx.fillStyle = data[i].color; this.ctx.fillRect(280, legendY, 20, 15); this.ctx.fillStyle = '#333'; this.ctx.font = '12px sans-serif'; this.ctx.fillText(data[i].name + ' ' + data[i].value, 310, legendY + 12); legendY += 25; } }

    🔧 实训四：动态刷新图表（15分钟）

  • 添加一个Button组件，点击后随机生成新的资产数据。
  • 在按钮点击回调中更新数据数组，重新调用绘制函数。
  • 注意： 每次绘制前需要清空画布（fillRect全白或clearRect）。
    • @State chartData: number[] = [42, 28, 35, 51, 19]; build() { Column() { Canvas(this.ctx) .width('100%') .height(400) .onReady(() => { this.drawChart(); }) Button('随机刷新数据') .onClick(() => { // 生成随机数据 this.chartData = this.chartData.map(() => Math.floor(Math.random() * 80) + 10); this.drawChart(); }) } }
