🧩 知识讲解 · 鸿蒙列表的魔法

1. List & ListItem —— 货架与商品

📦 List组件： 专门用来展示大量数据的滚动容器，高效渲染，支持懒加载。

ListItem： List中的每一个条目，里面可以放任何UI（文字、图片、按钮）。

ForEach： 循环渲染，根据数组数据批量生成ListItem。

// 最简单的列表结构 List() { ForEach(this.assetList, (item: AssetItem) => { ListItem() { Text(item.name) .fontSize(16) .padding(12) } }) }

2. 数据模型定义（资产对象）

// 定义资产数据接口 interface AssetItem { id: number; name: string; category: string; quantity: number; price: number; icon?: string; // 图标占位 } // 模拟数据（后续替换成真实数据） @State assetList: AssetItem[] = [ { id: 1, name: "笔记本电脑", category: "电子设备", quantity: 5, price: 5500 }, { id: 2, name: "投影仪", category: "教学设备", quantity: 2, price: 3200 }, { id: 3, name: "实训开发板", category: "实训器材", quantity: 20, price: 280 }, { id: 4, name: "交换机", category: "网络设备", quantity: 3, price: 1200 } ];

3. 列表项卡片布局（Row + Column）

每个资产卡片包含：左侧图标占位 + 中间名称/分类/数量 + 右侧价格/箭头。使用Flex弹性布局。

ListItem() { Row() { // 左侧图标占位 Image($r("app.media.ic_asset")) .width(48).height(48).margin({ right: 12 }) // 中间信息 Column() { Text(item.name).fontSize(16).fontWeight(FontWeight.Medium) Text(`${item.category} · 数量:${item.quantity}`) .fontSize(12).fontColor("#999") }.alignItems(HorizontalAlign.Start).layoutWeight(1) // 右侧价格 + 箭头 Column() { Text(`¥${item.price}`).fontSize(14).fontColor("#0078d4") Text("详情").fontSize(10).fontColor("#aaa") }.alignItems(HorizontalAlign.End) }.padding(12).width('100%') }

    🐛 常见Bug及解决 · 避坑指南

    ❌ 列表不显示

    原因：ForEach的key生成不正确或数据为空。解决：给ForEach第二个参数写(item) => item.id.toString()。

    ❌ 布局错位

    原因：Row/Column未设置宽度或layoutWeight。解决：中间信息Column添加.layoutWeight(1)自动撑开。

    ❌ 图片不显示

    原因：资源路径错误或未添加图片。解决：使用网络图片测试Image('https://xxx')或检查资源文件夹。
