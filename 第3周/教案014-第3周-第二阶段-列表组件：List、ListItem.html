🧩 知识讲解 · List组件 = 手机里的“无限滚动条”

1. 认识List和ListItem —— 像超市货架一样摆商品

🛒 趣味比喻： 如果把手机屏幕比作一个“货架”，List就是整个货架，ListItem就是货架上每个商品。商品可以有很多，用户上下滑动就能看到所有。

代码最小结构（伪代码）：

List() {
    ForEach(数据数组, (item) => {
        ListItem() {
            // 这里放每个条目的内容
            Text(item.name)
        }
    })
}

💡 关键点：ForEach是鸿蒙的循环指令，帮我们自动创建多个ListItem，不需要一个一个手写！

2. 数据从哪里来？ —— 先用假数据模拟

// 在组件内部定义资产数据数组
@State assets: Array<Asset> = [
    { id: 1, name: '笔记本电脑', count: 25, icon: '💻' },
    { id: 2, name: '投影仪', count: 8, icon: '📽️' },
    { id: 3, name: '实训开发板', count: 40, icon: '🔌' },
    { id: 4, name: '显示器', count: 15, icon: '🖥️' }
];

// 定义类型（方便理解）
interface Asset {
    id: number;
    name: string;
    count: number;
    icon: string;
}

📌 小贴士： 先写死数据让列表跑起来，以后学了数据库再替换成真实数据。

3. ListItem的“装修” —— 卡片式布局

📐 结构设计

每个ListItem内部用Row水平排列：左边放图标，中间放名称+数量，右边放一个“详情”箭头。

🎨 样式美化

给ListItem加背景色、圆角、边距，鼠标/手指按下去有高亮效果。

ListItem() {
    Row() {
        Text(item.icon).fontSize(28).margin({right: 12})
        Column() {
            Text(item.name).fontSize(18).fontWeight(FontWeight.Bold)
            Text(`数量: ${item.count}`).fontSize(14).fontColor('#666')
        }.alignItems(HorizontalAlign.Start)
        Blank()  // 占据中间空白
        Text('›').fontSize(24).fontColor('#0078d4')
    }
    .width('100%')
    .padding(12)
    .backgroundColor('#f8f9fa')
    .borderRadius(12)
    .margin({bottom: 8})
}
.onClick(() => {
    // 点击时弹窗或跳转
    AlertDialog.show({ message: `你点击了 ${item.name}` })
})

🔧 实训任务一：定义资产数据模型（5分钟）

  • 在文件顶部（@Entry之前）定义接口：interface Asset { id: number; name: string; count: number; icon: string; }
  • 在组件内用@State assets: Asset[] = [...]定义测试数据（参考上面示例）。
  • 验证： 在build()里随便加个Text显示数组长度，看是否编译通过。

    • 🔧 实训任务二：搭建List骨架（15分钟）

  • 在build()里，先放一个Column作为容器，设置padding(20)。
  • 添加标题Text(“📦 我的资产”， fontSize 24, fontWeight bold)。
  • 添加List组件，设置height('100%')，并给它一个背景色（方便观察）。
  • 在List里写ForEach(this.assets, (item: Asset) => { ListItem() { ... } }, (item) => item.id.toString())。
  • 重点： ForEach的第三个参数是唯一标识key，必须用item.id，否则列表刷新会出bug。
  • 运行预览器，观察是否有白色区域（列表区域）出现。

    • 💡 此时列表里还没内容，但是List已经存在，下一步我们“装修”每个ListItem。

    🔧 实训任务三：设计卡片式ListItem（25分钟）

  • 在ListItem的括号内，写Row组件，对齐方式设置.alignItems(VerticalAlign.Center)。
  • 左边：用Text显示item.icon，字体大小30，右边距12。
  • 中间：用Column包含两个Text：第一个显示名称（加粗），第二个显示“数量: X”。设置Column的alignItems为Start。
  • 右边：放一个箭头符号“›”，字体大小24，颜色蓝色。
  • 给Row添加样式：.width('100%').padding(12).backgroundColor('#f8f9fa').borderRadius(12).margin({bottom: 8})。
  • 在预览器中查看效果，上下滚动看是否流畅。
