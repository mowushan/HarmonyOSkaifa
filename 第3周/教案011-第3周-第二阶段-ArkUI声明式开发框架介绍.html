🧩 知识讲解 · 声明式UI的魔法世界

1. 声明式 vs 命令式 —— 做饭 vs 点外卖

🍳 趣味引入： 假设你要吃一顿饭——

  • 命令式（传统UI开发）： 去菜市场买菜 → 洗菜切菜 → 开火炒菜 → 装盘上桌。每一步都要亲自指挥。
  • 声明式（ArkUI）： 打开外卖App，点一份“宫保鸡丁”，坐等送达。你只关心“要什么”，不关心“怎么做”。

🎯 总结： 声明式UI = 描述界面长什么样 + 数据变化时自动更新，框架帮你完成所有“怎么做”的脏活累活！

2. ArkUI页面三件套 —— @Entry、@Component、build()

🏷️ @Entry

表示这是页面的入口，类似“大门”。每个应用至少有一个@Entry标记的组件。

🧩 @Component

表示这是一个自定义组件，你可以把它想象成一个“积木块”。

🏗️ build()

组件的“施工图”，里面写UI结构。所有界面元素都放在build()里。

// 最简单的ArkUI页面长这样：
@Entry
@Component
struct HelloPage {
    build() {
        Column() {
            Text('你好，鸿蒙！')
                .fontSize(30)
                .fontColor('#0078d4')
            Button('点我试试')
                .onClick(() => {
                    console.log('按钮被点击了')
                })
        }
        .width('100%')
        .height('100%')
    }
}

3. 状态驱动UI —— @State 魔法变量

✨ 核心魔法： 给变量加上@State，当变量值改变时，用到这个变量的所有界面会自动刷新！

传统写法： 改变数据 → 手动找到UI元素 → 手动更新文字（麻烦易错）
ArkUI写法： 改变@State变量 → 界面自动变（优雅省心）

// 状态变量示例：点击按钮数字+1
@Entry
@Component
struct Counter {
    @State count: number = 0;   // 加上@State，变成“魔法变量”

    build() {
        Column() {
            Text(`点击次数: ${this.count}`)
                .fontSize(24)
            Button('点我+1')
                .onClick(() => {
                    this.count++;   // 改变变量，界面自动刷新！
                })
        }
    }
}

    🧪 实训任务二：添加Button组件（15分钟）

  • 在Text组件下面（同一Column内）添加Button组件：
    • Button('点我试试') .onClick(() => { console.log('按钮被点击了') })
  • 运行预览： 点击按钮，打开底部“Log”窗口，看看有没有输出日志。
  • 改造一下： 把按钮文字改成 "惊喜按钮"，点击时弹出一个提示框（Toast）。提示：可以用 promptAction.showToast({ message: '你点到了惊喜！' })。
  • 注意： 需要在文件顶部导入 import promptAction from '@ohos.promptAction';

    • 🔔 常见问题： 如果Button不显示，检查是否放在了Column的{}里面，且没有覆盖或超出屏幕。

    🧪 实训任务三：实现点击计数器（20分钟）—— 核心实战！

  • 声明状态变量： 在struct里面，build()上面添加：
    @State count: number = 0;
  • 修改Text组件： 让它显示点击次数 Text(`点击了 ${this.count} 次`)。
  • 修改Button的点击事件： 把原来的代码改成 this.count++;。
  • 运行并测试： 每次点击按钮，数字应该自动增加！
  • ✨ 增强挑战： 添加第二个按钮“重置”，点击后让count变回0。
    • // 完整计数器代码示例：
@Entry
@Component
struct CounterDemo {
    @State count: number = 0;

    build() {
        Column({ space: 20 }) {
            Text(`点击次数: ${this.count}`)
                .fontSize(32)
                .fontWeight(FontWeight.Bold)
            
            Button('点我 +1')
                .onClick(() => {
                    this.count++;
                })
            
            Button('重置清零')
                .backgroundColor('#FF6600')
                .onClick(() => {
                    this.count = 0;
                })
        }
        .width('100%')
        .height('100%')
        .justifyContent(FlexAlign.Center)
    }
}
