🧩 知识讲解 · 认识三大组件

1. Text组件 —— 让文字“说话”

📝 趣味比喻： Text就像一张张“文字便利贴”，你可以贴在屏幕的任何位置，还能改变它的样子！

常用属性（像给文字化妆）：

  • fontSize(20) → 字号大小，数字越大字越大
  • fontColor(Color.Red) → 文字颜色，红橙黄绿青蓝紫随便选
  • fontWeight(FontWeight.Bold) → 加粗，让文字更有力量
  • textAlign(TextAlign.Center) → 居中、左对齐、右对齐
// Text组件的基本写法
Text('欢迎来到鸿蒙世界')
  .fontSize(28)
  .fontColor(Color.Blue)
  .fontWeight(FontWeight.Bold)
  .textAlign(TextAlign.Center)
  .width('100%')

2. Button组件 —— 点击就有反应

🔘 趣味比喻： 按钮就像游戏里的“触发开关”，按下去就会触发魔法！

核心玩法： onClick(() => { 点击后要做的事情 })

样式美化： backgroundColor背景色，borderRadius圆角，padding内边距。

// 按钮 + 点击事件
Button('点我试试')
  .onClick(() => {
    console.log('按钮被点击了！')
  })
  .backgroundColor('#0078d4')
  .borderRadius(20)
  .padding({ left: 20, right: 20, top: 10, bottom: 10 })

3. Image组件 —— 一图胜千言

🖼️ 趣味比喻： Image就像相框，把图片装进去展示出来。

两种图片来源：

  • 📁 本地图片：$r('app.media.icon') 或 $rawfile('logo.png')
  • 🌐 网络图片：src('https://xxx.com/pic.jpg')（需要申请网络权限）

常用设置： width/width高度，objectFit缩放模式（Cover保持比例填满，Contain完整显示）。

// 图片组件示例
Image($r('app.media.startIcon'))
  .width(100)
  .height(100)
  .objectFit(ImageFit.Cover)
  .borderRadius(16)

⚙️ 技能实训 · 动手搭建“个人名片”界面

✅ 实训目标：综合运用Text、Button、Image，制作一个可交互的“自我介绍卡片”

📁 课前准备

1. 打开DevEco Studio，新建一个Empty Ability项目，取名 UIComponentsDemo。

2. 准备好一张本地图片（可以是自己的照片或喜欢的logo），复制到 entry/src/main/resources/base/media/ 文件夹下，命名为 avatar.jpg。

3. 我们将在 Index.ets 中编写代码。

🧪 实训任务一：搭建基础布局（15分钟）

  • 编写基础结构： 在 build() 中创建 Column 容器，设置居中。
  • 添加头像图片： 使用Image组件加载刚刚放入的图片，设置宽100、高100，圆角50（变成圆形）。
  • 添加姓名Text： 显示“张小明”（替换自己的名字），字体24号，加粗，深蓝色。
  • 添加职位Text： 显示“鸿蒙开发者 · 预备队员”，字体16号，灰色。
  • 运行预览： 在预览器或模拟器中查看效果，调整间距。
    • // 基础布局代码示例
Column({ space: 12 }) {
  Image($r('app.media.avatar'))
    .width(100)
    .height(100)
    .borderRadius(50)
  Text('张小明')
    .fontSize(24)
    .fontWeight(FontWeight.Bold)
    .fontColor('#1e466e')
  Text('鸿蒙开发者 · 预备队员')
    .fontSize(16)
    .fontColor('#7f8c8d')
}
.width('100%')
.height('100%')
.justifyContent(FlexAlign.Center)
.alignItems(HorizontalAlign.Center)

    🧪 实训任务二：添加交互按钮（20分钟）

  • 添加“打招呼”按钮： 在Text下方添加Button，文字为“👋 打个招呼”。
  • 定义状态变量： 在组件顶部定义 @State message: string = '你好，很高兴认识你！'。
  • 点击按钮更新文字： 在按钮的onClick中，修改message的内容（例如改成“鸿蒙集训，未来可期！”）。
  • 添加显示消息的Text： 在按钮下方添加一个Text组件，内容绑定 this.message，字体颜色蓝色。
  • 测试交互： 点击按钮，下方文字会实时变化！
    • // 状态变量 + 按钮交互
@State message: string = '你好，很高兴认识你！'

// 在Column中添加
Button('👋 打个招呼')
  .onClick(() => {
    this.message = '鸿蒙集训，未来可期！🚀'
  })
  .backgroundColor('#0078d4')
  .fontColor(Color.White)
  .borderRadius(20)
  .padding({ left: 20, right: 20, top: 8, bottom: 8 })

Text(this.message)
  .fontSize(16)
  .fontColor('#0078d4')
  .margin({ top: 16 })

    🧪 实训任务三：图片切换（25分钟）

  • 准备多张图片： 再添加一张图片到media文件夹，比如 avatar2.jpg。
  • 定义图片资源变量： @State currentAvatar: Resource = $r('app.media.avatar')。
  • 添加“换头像”按钮： 添加第二个按钮，文字为“🔄 换个头像”。
  • 点击切换图片： 在onClick中判断当前头像，切换成另一张。
  • 美化按钮： 设置不同背景色，两个按钮并排显示（使用Row组件包裹）。
    • // 图片切换逻辑
@State currentAvatar: Resource = $r('app.media.avatar')
@State isFirst: boolean = true

Image(this.currentAvatar)
  .width(100)
  .height(100)
  .borderRadius(50)

Row({ space: 20 }) {
  Button('👋 打招呼')
    .onClick(() => { this.message = '鸿蒙集训，未来可期！🚀' })
  Button('🔄 换头像')
    .onClick(() => {
      if (this.isFirst) {
        this.currentAvatar = $r('app.media.avatar2')
      } else {
        this.currentAvatar = $r('app.media.avatar')
      }
      this.isFirst = !this.isFirst
    })
}

    🧪 实训任务四：综合挑战 · 完成“个人名片”完整版（20分钟）

    🎯 最终成品要求：

    1. 顶部圆形头像（可点击换头像）
    2. 姓名 + 格言/职位（用Text组件）
    3. 一个计数按钮：点击后显示“已经点击X次”，记录点击次数
    4. 一个“重置”按钮，将计数归零，同时把消息恢复默认
    5. 整体布局优雅，有适当的边距和圆角

    💡 提示： 使用@State count: number = 0来记录次数，按钮里this.count++。

    // 完整版核心代码片段
@State count: number = 0

Text(`点击次数：${this.count}`)
  .fontSize(18)
  .margin({ top: 16 })

Row({ space: 20 }) {
  Button('👍 点赞')
    .onClick(() => { this.count++ })
  Button('🗑️ 重置')
    .onClick(() => { 
      this.count = 0
      this.message = '你好，很高兴认识你！'
    })
}
