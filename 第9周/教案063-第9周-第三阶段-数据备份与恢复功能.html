3. 备份核心原理：复制文件

// 复制文件函数（同步封装）
async copyFile(src: string, dest: string): Promise<void> {
    try {
        let stat = await fileio.stat(src);
        let buf = new ArrayBuffer(stat.size);
        let fdSrc = await fileio.open(src, 0o0);
        await fileio.read(fdSrc, buf);
        await fileio.close(fdSrc);
        
        let fdDest = await fileio.open(dest, 0o2 | 0o100, 0o666);
        await fileio.write(fdDest, buf);
        await fileio.close(fdDest);
        console.info("备份成功：" + dest);
    } catch (e) {
        console.error("备份失败：" + JSON.stringify(e));
    }
}

    🔧 实训二：编写备份核心逻辑

  • 获取数据库源文件路径：
  • 生成备份文件名： 建议格式 asset_backup_20250412_143022.db（含时间戳）
  • 选择备份目录： 使用 fileManager.getExternalStorageDir() 获取公共Download目录（需申请读写权限！）
  • 执行文件复制，并弹窗提示成功或失败。
    • async backupDatabase() {
    try {
        let context = getContext(this);
        let srcFile = context.databaseDir + '/my_asset.db';
        let stat = await fileio.stat(srcFile);
        if (stat.size === 0) {
            prompt.showToast({ message: '没有数据可备份' });
            return;
        }
        
        // 生成带时间戳的文件名
        let now = new Date();
        let timestamp = `${now.getFullYear()}${now.getMonth()+1}${now.getDate()}_${now.getHours()}${now.getMinutes()}${now.getSeconds()}`;
        let fileName = `asset_backup_${timestamp}.db`;
        
        // 公共Download目录
        let backupDir = fileManager.getExternalStorageDir() + '/Download/';
        let destFile = backupDir + fileName;
        
        await this.copyFile(srcFile, destFile);
        prompt.showToast({ message: `备份成功！\n位置：下载目录` });
    } catch(e) {
        prompt.showToast({ message: '备份失败，请检查权限' });
        console.error(JSON.stringify(e));
    }
}

    🔧 实训三：恢复数据 + 二次确认

  • 弹出对话框，让用户选择备份文件： 扫描下载目录下所有 .db 文件。
  • 确认恢复前，提示当前数据将被覆盖！
  • 执行恢复： 关闭数据库连接 → 复制备份文件覆盖原数据库 → 重新打开数据库 → 刷新界面列表。
    • async confirmRestore() {
    AlertDialog.show({

            action: () => {}

async restoreDatabase() {
    // 简易版本：选择最新备份文件（或弹出文件选择器）
    let backupDir = fileManager.getExternalStorageDir() + '/Download/';
    // 此处可以扫描该目录下所有asset_backup_*.db文件，展示列表供用户选择
    // 为简化先取第一个
    let files = await fileio.listFile(backupDir);
    let backupFile = files.find(f => f.includes('asset_backup_') && f.endsWith('.db'));
    if (!backupFile) {
        prompt.showToast({ message: '未找到备份文件' });
        return;
    }
    let srcBackup = backupDir + backupFile;
    let context = getContext(this);
    let destDb = context.databaseDir + '/my_asset.db';
    
    // 关闭数据库（如有打开的连接先close）
    await this.closeDatabase();
    // 复制备份到数据库目录
    await this.copyFile(srcBackup, destDb);
    // 重新初始化数据库
    await this.initDatabase();
    // 刷新列表
    this.loadAssetList();
    prompt.showToast({ message: '恢复成功！' });
}
