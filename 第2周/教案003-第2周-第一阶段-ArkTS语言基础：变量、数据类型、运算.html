🧩 知识讲解 · 让程序学会“记东西”

1. 变量 —— 程序的“记忆盒子”

📦 趣味比喻： 想象你要给朋友送礼物，你会先把礼物放进一个盒子，盒子上写个名字“给张三的礼物”。程序里的变量就是这个盒子！

ArkTS中声明变量的两种方式：

  • let 变量名 = 值; —— 可以变化的盒子，里面的东西可以换。
  • const 变量名 = 值; —— 封死的盒子，里面的东西永远不变。
// 声明一个可变变量
let studentName = "王小萌";
studentName = "李大力";   // 可以改，没问题！

// 声明一个常量（不变的值）
const SCHOOL = "广东省粤东技师学院";
// SCHOOL = "其他学校";   // ❌ 这行会报错！常量不能改

起名规则（很重要！）： 只能用字母、数字、下划线，不能用数字开头，不能是关键字（如let、const）。
✅ 好名字: studentAge、phoneNumber、totalScore
❌ 坏名字: 2score（数字开头）、let（关键字）

2. 数据类型 —— 盒子里的“不同物品”

🔢 数字(number)

整数或小数。
let age = 18;
let price = 99.9;

📝 字符串(string)

文本，用引号包起来。
let name = "张三";
let msg = 'Hello';

✅ 布尔值(boolean)

只有两个值：true（真）和false（假）。
let isPass = true;
let hasPhone = false;

// 数据类型演示
let myNumber = 42;
let myText = "鸿蒙开发真有趣！";
let myBool = true;

console.log(typeof myNumber);   // 输出: number
console.log(typeof myText);     // 输出: string
console.log(typeof myBool);     // 输出: boolean

3. 运算符 —— 让数据“动起来”

类型符号含义例子
算术运算符+加法3 + 5 → 8
-减法10 - 4 → 6
*乘法6 * 7 → 42
/除法15 / 3 → 5
%取余（模）10 % 3 → 1
赋值运算符=赋值let x = 10;
+=加后赋值x += 5 → x = x + 5
-=减后赋值x -= 2 → x = x - 2
 
// 运算符实战
let a = 15;
let b = 4;

console.log(a + b);   // 19
console.log(a - b);   // 11
console.log(a * b);   // 60
console.log(a / b);   // 3.75
console.log(a % b);   // 3（15除以4余3）

let score = 80;
score += 10;          // score变成了90
console.log(score);   // 90

    🐛 常见错误 · 别怕，老师教你解决

    ❌ 变量未定义

    Cannot find name 'xxx'
    ➡️ 检查变量名是否拼写错误，是否忘记用let声明。

    ❌ 常量重新赋值

    Assignment to constant variable
    ➡️ const声明的变量不能改，改用let。

    ❌ 类型不匹配

    Operator '+' cannot be applied
    ➡️ 确保运算符两边类型一致，或使用toString()转换。

    🛠️ 调试小技巧： 遇到红色波浪线，鼠标放上去看提示；用console.log打印中间值，看哪里出问题。
