🧩 知识讲解 · 让代码学会“思考”和“分工”

1. 条件判断 if-else —— 程序的“红绿灯”

🚦 趣味场景： 如果你考了90分以上，爸妈奖励你；如果考了60-89分，不奖不罚；如果低于60分，要写检讨！—— 这就是if-else if-else的逻辑。

// ArkTS 语法示例 (类似TypeScript)
let score: number = 85;
if (score >= 90) {
    console.log("🎉 太棒了！奖励鸿蒙开发板！");
} else if (score >= 60) {
    console.log("👍 不错，继续努力！");
} else {
    console.log("📚 别灰心，我们一起复习！");
}

📌 关键点： 条件必须是布尔表达式（true/false）；`else if` 可以有多个；`else` 可选。

2. 循环 for —— 让电脑当“复读机”

🔄 for循环格式

for (初始值; 条件; 变化) { 循环体 }

例如：for (let i = 1; i <= 5; i++) 表示 i从1到5，每次+1。

📊 实用案例

计算1加到100：
let sum = 0;
for (let i = 1; i <= 100; i++) { sum += i; }

结果：5050

// 打印1~10每个数的平方
for (let num = 1; num <= 10; num++) {
    console.log(`${num} 的平方是 ${num * num}`);
}

3. 函数 —— 制造“代码积木”

🧩 为什么要用函数？ 假如你要多次计算两个数的和，不用函数就得重复写相加代码；用函数，定义一次，随时调用。

// 定义函数
function greet(name: string): void {
    console.log(`你好，${name}！欢迎来到鸿蒙世界！`);
}

function add(a: number, b: number): number {
    return a + b;
}

// 调用函数
greet("王嘉鑫");
let result = add(15, 27);
console.log(`15+27=${result}`);

✅ 函数三要素： 函数名、参数（输入）、返回值（输出）。无返回值用void。

    🧪 实训任务四：综合项目 —— “智能成绩评定系统” （20分钟）

    项目需求： 输入一个学生的分数（0-100），输出等级并给出评语，同时累计全班平均分。

    功能点：

    1. 编写函数 getGrade(score) 返回“优秀(90-100)”“良好(75-89)”“及格(60-74)”“不及格(<60)”。
    2. 编写函数 getComment(grade) 返回鼓励语句。
    3. 模拟一个班级5个学生的分数（预先定义数组），用循环调用函数，输出每个学生的等级和评语，同时计算平均分。
    4. 最后输出“班级平均分:xx，最高分是xx”。
    // 参考框架
let scores: number[] = [95, 82, 67, 45, 88];
let total = 0;
let maxScore = 0;
for (let s of scores) {
    let grade = getGrade(s);
    let comment = getComment(grade);
    console.log(`分数 ${s} → ${grade}，${comment}`);
    total += s;
    if (s > maxScore) maxScore = s;
}
console.log(`平均分: ${total / scores.length}, 最高分: ${maxScore}`);
