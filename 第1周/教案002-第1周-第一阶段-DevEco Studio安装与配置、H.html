🧩 知识讲解 · 鸿蒙系统架构全景图

1. 鸿蒙系统架构 —— 一图看懂鸿蒙

📊 鸿蒙系统分层架构（简化版）

┌─────────────────────────────────┐

│ 📱 应用层 (App) │ ← 我们写的应用

├─────────────────────────────────┤

│ 🧩 框架层 (ArkUI, Ability) │ ← 提供UI组件、页面管理

├─────────────────────────────────┤

│ 🔧 系统服务层 │ ← 分布式调度、元服务、通知

├─────────────────────────────────┤

│ 🌐 内核层 (Linux Kernel) │ ← 基础系统核心

└─────────────────────────────────┘

✨ 最关键的技术：分布式软总线 —— 像一根“看不见的数据线”，让多个设备无缝连接、协同工作！

🎯 趣味类比： 鸿蒙系统就像一个“智能小区”——

  • 🏢 内核层 = 地基和钢筋水泥（稳固基础）
  • ⚡ 系统服务层 = 水电燃气管道（提供基础服务）
  • 🏠 框架层 = 房间户型设计（让开发者方便建房子）
  • 🛋️ 应用层 = 家具和装饰（我们做的App）
  • 🔌 分布式软总线 = 小区里的智能快递通道，让不同楼栋（设备）互通！

2. DevEco Studio —— 鸿蒙开发者的“超级工具箱”

📦 代码编辑器

智能提示、代码高亮、自动补全，写代码快人一步。

📱 模拟器管理

无需真机，用电脑模拟手机/平板运行应用。

🐛 调试神器

断点调试、日志输出、性能分析，找bug不再痛苦。

🔌 插件生态

支持Git、代码格式化、AI辅助等扩展。

⚠️ 版本提醒： 请务必安装 4.0 及以上版本（建议最新稳定版），才能支持HarmonyOS NEXT开发特性。

⚙️ 技能实训 · 手把手安装开发环境

✅ 本次实训目标：每人电脑成功运行DevEco Studio，创建模拟器

🔧 实训一：下载DevEco Studio安装包

  • 打开华为开发者联盟官网： https://developer.huawei.com/consumer/cn/，点击顶部导航“开发” → “DevEco Studio”。
  • 选择对应版本： 根据你的操作系统（Windows x64 / macOS）点击下载。推荐下载 “Windows (64-bit)” 版本。
  • 保存安装包： 将下载的 deveco-studio-xxxx.exe 保存到 D:\鸿蒙安装包\ 文件夹（方便查找）。
  • 📸 打卡： 下载完成后，截图显示“下载完成”窗口，发到集训队群，老师统计进度。

    • ⚠️ 注意： 安装包约1.2GB，下载需要5-15分钟。如果网络慢，老师会共享U盘安装包，大家互相帮助传递。

    🔧 实训二：安装DevEco Studio（详细步骤）

  • 双击安装包： 以管理员身份运行（右键→以管理员身份运行）。
  • 选择安装路径： 建议修改为 D:\DevEcoStudio（不要放C盘，防止空间不足）。
  • 勾选组件： 全部保持默认勾选（创建桌面快捷方式、添加到PATH等）。
  • 等待安装完成： 大约需要3-5分钟，期间不要关闭安装窗口。
  • 启动DevEco Studio： 安装完成后，勾选“Run DevEco Studio”，点击Finish。
  • 首次启动配置： 导入设置选择“Do not import settings”，然后点击OK。
  • 同意许可协议： 阅读后点击“Accept”，等待初始化完成。

    • ✅ 验证安装成功： 看到欢迎界面，左上角显示“DevEco Studio”，就说明安装成功啦！

    🔧 实训三：配置HarmonyOS SDK（核心步骤）

  • 打开SDK Manager： 点击欢迎界面右下角的“Configure” → “SDK Manager”。
  • 设置SDK路径： 建议修改为 D:\DevEcoStudio\sdk，点击“Edit”更改。
  • 下载API版本： 在“SDK Platforms”选项卡中，勾选 “HarmonyOS API 9+” 或最新稳定版本，点击“Apply”下载。
  • 等待下载： 大约需要5-10分钟，进度条走完即可。
  • 验证SDK： 在SDK Manager的“SDK Tools”中，确认“Emulator”、“Toolchains”都已勾选。

    • 💡 如果下载速度慢，可以配置国内镜像（华为镜像、阿里云镜像），老师现场指导修改。

    🔧 实训四：创建本地模拟器（预备运行环境）

  • 打开设备管理器： 在DevEco Studio欢迎界面，点击“More Actions” → “Device Manager”。
  • 创建模拟器： 点击右上角“+”号，选择“Phone”设备类型。
  • 选择系统镜像： 选择一个API版本（例如API 9），点击“Next”。
  • 配置模拟器： 给模拟器起名“鸿蒙学习机”，分辨率默认即可，点击“Finish”。
  • 启动模拟器： 在设备列表中，点击绿色三角按钮启动模拟器。
  • 等待开机： 第一次启动较慢（约2-3分钟），看到手机界面就成功啦！

    • 🎉 恭喜！ 你已经完成了鸿蒙开发环境搭建！现在你的电脑已经变成“鸿蒙开发工作站”啦！

    🔧 常见问题 · 急救指南

    ❓ 提示“JDK未找到”

    安装包自带JDK，重新启动安装程序，勾选“Install JDK”即可。

    ❓ 模拟器启动黑屏

    检查电脑是否开启虚拟化（BIOS开启Intel VT-x），或者改用远程模拟器。

    ❓ 下载SDK太慢

    在SDK Manager中点击“Settings”，设置HTTP Proxy为华为镜像：https://mirrors.huaweicloud.com/

    ❓ 安装路径有中文

    DevEco Studio路径不要包含中文或空格，建议D:\DevEcoStudio。

    📞 遇到问题不要慌！举手示意，老师会带“移动U盘急救包”帮你解决。课后也可以随时在集训群提问。
