🧩 知识讲解 · 鸿蒙江湖与大赛宝藏

1. 鸿蒙（HarmonyOS）—— 不只是手机系统

📱+💻+⌚ 趣味小剧场： 假如你的手机在播放视频，轻轻一划，视频就能“飞”到平板上继续播放；手机上的导航，手表上自动同步路线……这就是鸿蒙“分布式”能力！

三个关键词帮你记住鸿蒙：

  • 🌐 全场景：一套代码，适配手机、平板、智慧屏、手表等。
  • 🔗 超级终端：多个设备“融合”成一个虚拟大终端。
  • ⚡ 原子化服务：不用安装，即用即走（卡片体验）。

📣 参加鸿蒙生态开发大赛，就是学习如何用这些酷炫能力解决生活中的问题！

2. 华为开发者大赛 · 赛道与评分标准

🏅 应用创新赛道

基于鸿蒙开发实用App，如智慧校园、资产管理、健康助手等。我们集训队重点攻这个！

🏅 元服务赛道

开发轻量化卡片服务，用户无需下载，轻量体验。适合做快捷工具。

🎯 评分关键点

创新性(30%) + 鸿蒙特性应用(30%) + 完整度(20%) + 用户体验(20%)。

✅ 我们的目标： 做出一个能跑在手机+平板上，具备数据管理和鸿蒙特有功能（服务卡片或分布式协同）的参赛作品，争取省级以上奖项！

⚙️ 技能实训 · 今日动手任务清单

✅ 本次实训目标：完成华为账号注册、熟悉官网、制定团队公约

🔧 实训一：华为开发者联盟账号注册（全员必做）

  • 打开浏览器，访问华为开发者联盟官网：https://developer.huawei.com/consumer/cn/
  • 点击右上角“登录”，选择“注册”。按提示输入手机号或邮箱，获取验证码，设置密码。
  • 实名认证（可选，建议老师协助）：若学生已满16岁可用身份证实名，未满可由老师统一辅助认证。集训队可以先用账号，后续统一补充。
  • 登录成功后，在个人中心找到“我的应用”或“开发服务”，确认可以访问“AppGallery Connect”等模块。
  • 📸 截图打卡： 把登录成功的页面截图，发到集训队群，老师一一核对。

    • 💡 为什么必须注册？ 下载DevEco Studio、查阅开发文档、将来上架应用都需要华为账号。现在注册，下节课直接开练！

    🔧 实训二：探索鸿蒙官方资源 · 发现宝藏

  • 打开华为开发者学堂：https://developer.huawei.com/consumer/cn/training/
  • 在搜索框输入“鸿蒙第一课”，找到免费课程并添加到“我的学习”。
  • 找到“Codelabs”标签页，浏览一下“构建你的第一个HarmonyOS应用”的简介（不用真做，只是熟悉入口）。
  • 📌 小组任务： 2人一组，互相讨论“你觉得鸿蒙能做什么有趣的应用？” 每组写一个点子（比如：教室座位管理系统、设备巡检助手）并写在便利贴上，课代表收集。

    • 📚 华为开发者学堂是我们最重要的免费学习平台，所有官方视频、Codelabs都在这里。下节课我们将从这里开始第一个代码实战！

    🔧 实训三：组建备赛战队 & 制定团队公约

    👥 战队组建（3-4人/组）

    原则：组长1名（负责督促进度），技术员2名，文档员1名。建议性格互补，最好有“代码狂热分子”+“细心整理员”。

    📜 团队公约（共创）

    1. 每周至少线下讨论2次；
    2. 每日代码上传Git（后续教）；
    3. 遇到bug先自己尝试解决10分钟，再求助组员；
    4. 互相鼓励，不嘲笑“小白问题”。

  • 分组讨论（15分钟）： 各组内自我介绍，推选组长，给战队起名（如“鸿蒙先锋队”、“星河舰队”）。
  • 共同写下公约： 用手机便签或纸笔记录3-5条团队规则，组长在班级群里上传。
  • 宣誓仪式： 每组组长上台介绍战队名和公约，全班互相监督。

    • 🎉 老师巡场：确保每位同学都有账号，并加入集训队班级群（企业微信/钉钉）。约定好每周作业提醒方式。

    🔧 实训四（拓展）：开发环境预览 · 下载DevEco Studio（可选）

    💻 如果教室网络条件允许，可以直接下载DevEco Studio安装包（windows版），下节课直接安装。

    下载指引： 华为开发者联盟官网 → 开发 → DevEco Studio → 下载Windows版本（约1.2GB）。

    👉 无法下载的同学也不用担心，老师会把安装包共享到班级网盘，下节课统一安装。
