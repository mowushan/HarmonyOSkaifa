🧩 知识讲解 · 菜单与操作选项

1. 菜单组件 Menu — 让用户“点一点就有选择”

🍔 趣味引入： 大家用微信的时候，长按一条消息，是不是会弹出“复制、转发、删除”？这就是菜单！鸿蒙里我们可以用 Menu 组件实现。

两种使用方式：

  • 方式一：bindMenu — 给任意组件绑定菜单，点击或长按时弹出。
  • 方式二：Menu组件 — 单独定义菜单内容，灵活控制显示时机。
// 方式一：使用 bindMenu 绑定菜单
@Entry
@Component
struct MenuDemo {
  build() {
    Column() {
      Text('长按我试试')
        .fontSize(20)
        .padding(20)
        .backgroundColor('#f0f0f0')
        .bindMenu([
          { value: '编辑', action: () => { console.log('编辑') } },
          { value: '删除', action: () => { console.log('删除') } },
          { value: '分享', action: () => { console.log('分享') } }
        ])
    }
    .width('100%')
    .height('100%')
  }
}

2. 自定义菜单 — 更灵活的样式

// 方式二：使用 @Builder 自定义菜单样式
@Builder
function MyMenu() {
  Menu() {
    MenuItem({ content: '编辑' })
      .onClick(() => { console.log('编辑') })
    MenuItem({ content: '删除' })
      .onClick(() => { console.log('删除') })
    MenuItem({ content: '分享' })
      .onClick(() => { console.log('分享') })
  }
}

@Entry
@Component
struct CustomMenuDemo {
  @State showMenu: boolean = false

  build() {
    Column() {
      Button('点我弹出菜单')
        .onClick(() => { this.showMenu = true })
        .bindMenu(this.showMenu, MyMenu())
    }
  }
}

3. 底部操作面板 ActionSheet — 多选项的优雅方案

📱 当选项超过3个时，底部弹出的 ActionSheet 比菜单更合适，比如“选择排序方式”、“选择筛选条件”。

// ActionSheet 示例
@Entry
@Component
struct ActionSheetDemo {
  @State showSheet: boolean = false

  build() {
    Column() {
      Button('更多操作')
        .onClick(() => {
          this.showSheet = true
        })
        .actionSheet({
          title: '请选择操作',
          message: '你可以对当前资产执行以下操作',
          confirm: { value: '取消', action: () => { console.log('取消') } },
          sheets: [
            { title: '编辑资产', action: () => { console.log('编辑') } },
            { title: '删除资产', action: () => { console.log('删除') } },
            { title: '复制资产', action: () => { console.log('复制') } },
            { title: '导出数据', action: () => { console.log('导出') } }
          ]
        })
    }
  }
}

⚙️ 技能实训 · 动手实现菜单功能

✅ 本次实训目标：为资产管理列表页添加长按菜单和底部操作面板

🔧 实训一：创建基础资产列表（回顾与准备）

  • 打开 DevEco Studio，在之前项目 AssetManager 中，找到 AssetList.ets 文件。
  • 确保有基础数据模型：定义一个简单的资产类，包含名称、金额、日期。
  • 编写简单列表：用 List + ForEach 展示资产数据，每个资产项是一个 ListItem。
    • // 数据模型
interface Asset {
  id: number
  name: string
  amount: number
  date: string
}

// 示例数据
@State assets: Asset[] = [
  { id: 1, name: '笔记本电脑', amount: 5999, date: '2025-03-01' },
  { id: 2, name: '显示器', amount: 1299, date: '2025-03-05' },
  { id: 3, name: '键盘', amount: 299, date: '2025-03-10' }
]

    🔧 实训二：给资产项添加长按菜单（bindMenu）

  • 修改 ListItem，添加 .bindMenu() 方法。
  • 定义菜单项：包含“编辑”、“删除”、“查看详情”。
  • 实现删除逻辑：点击删除时，从 assets 数组中移除该项。
    • // 在 ListItem 上添加 bindMenu
ListItem() {
  Column() {
    Text(asset.name).fontSize(18).fontWeight(FontWeight.Bold)
    Text(`金额: ¥${asset.amount}`).fontSize(14).fontColor('#666')
    Text(`日期: ${asset.date}`).fontSize(12).fontColor('#999')
  }
  .padding(12)
  .width('100%')
}
.bindMenu([
  {
    value: '编辑',
    action: () => {
      console.log(`编辑资产: ${asset.name}`)
      // 跳转到编辑页面
    }
  },
  {
    value: '删除',
    action: () => {
      // 删除当前资产
      let index = this.assets.findIndex(item => item.id === asset.id)
      if (index !== -1) {
        this.assets.splice(index, 1)
      }
      prompt.showToast({ message: '删除成功' })
    }
  },
  {
    value: '查看详情',
    action: () => {
      console.log(`查看详情: ${asset.name}`)
    }
  }
])

    🔧 实训三：添加底部操作面板（ActionSheet）

  • 在页面顶部添加“更多操作”按钮，点击弹出 ActionSheet。
  • 定义操作选项：批量删除、导出数据、排序方式等。
  • 实现排序功能：按金额排序或按日期排序。
    • // 在 build() 方法中添加按钮
Button('更多操作')
  .margin(12)
  .onClick(() => {
    this.showMoreSheet = true
  })
  .actionSheet({
    title: '批量操作',
    message: '请选择要执行的操作',
    confirm: { value: '取消', action: () => { } },
    sheets: [
      {
        title: '按金额升序',
        action: () => {
          this.assets.sort((a, b) => a.amount - b.amount)
        }
      },
      {
        title: '按金额降序',
        action: () => {
          this.assets.sort((a, b) => b.amount - a.amount)
        }
      },
      {
        title: '按日期排序',
        action: () => {
          this.assets.sort((a, b) => a.date.localeCompare(b.date))
        }
      },
      {
        title: '导出数据',
        action: () => {
          prompt.showToast({ message: '导出功能开发中' })
        }
      }
    ]
  })

    🐛 常见错误 · 避坑指南

    ❌ 错误1：菜单不弹出

    原因： 忘记给组件添加 .bindMenu() 或点击区域太小。

    ✅ 解决：检查组件是否有足够的点击区域（加padding）。

    ❌ 错误2：删除后界面不更新

    原因： 数组操作没有触发UI刷新。

    ✅ 解决：使用 @State 装饰数组，使用 splice 或重新赋值。

    ❌ 错误3：ActionSheet 不显示

    原因： 没有绑定 .actionSheet() 或状态变量控制错误。

    ✅ 解决：确保 @State showSheet: boolean = false 并在点击时设为 true。
