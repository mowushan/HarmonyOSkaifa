🧩 知识讲解 · 让界面“活”起来的媒体组件

1. Image组件 —— 图片显示神器

🖼️ 趣味引入： 没有图片的App就像没有表情的朋友聊天，干巴巴的。Image组件就是给App“配图”的魔法师！

Image组件基本用法：

// 加载本地图片（放在 resources/base/media 目录下） Image($r('app.media.ic_asset')) // 引用资源文件 .width(100) .height(100) .borderRadius(12) // 加载网络图片 Image('https://example.com/photo.jpg') .width('100%') .height(200) .objectFit(ImageFit.Cover) // 裁剪模式：Cover / Contain / Fill

常用属性： width, height, borderRadius, objectFit (图片填充方式), alt (加载失败占位图)。

2. Video组件 —— 嵌入式播放器

🎥 类比： Video组件就像在手机里嵌了一个小电视，可以播放视频教程、产品宣传片。

Video({ src: 'https://example.com/demo.mp4', // 视频源 controller: this.videoController // 播放控制器 }) .width('100%') .height(200) .autoPlay(false) // 是否自动播放 .controls(true) // 显示控制条（播放/暂停/进度） .onStart(() => { console.log('播放开始'); }) .onPause(() => { console.log('暂停'); })

常用方法： controller.start() / pause() / stop() 等。

3. 网络权限配置（关键一步！）

🌐 鸿蒙应用访问网络图片或视频，必须在 module.json5 中申请网络权限：

"requestPermissions": [ { "name": "ohos.permission.INTERNET" } ]

否则图片和视频都加载不出来！等下实训会带大家一起配置。

⚙️ 技能实训 · 手把手打造媒体展示页面

✅ 实训目标：创建“媒体展示”页面，显示设备图片和操作视频

🔧 实训一：项目准备 & 权限配置（10分钟）

  • 打开已有的鸿蒙项目（或新建一个Empty Ability项目）。
  • 找到 entry/src/main/module.json5，在 "requestPermissions" 中添加网络权限（若没有则创建）。
  • 在 resources/base/media 文件夹下放入一张测试图片（比如 asset_demo.png），没有图片可用emoji占位。
  • 新建页面 MediaShowPage.ets，用于本次实训。

    • 💡 老师演示： 用投影展示如何修改module.json5，强调json格式正确性（逗号不能漏）。

    🔧 实训二：Image组件初体验 —— 加载本地图片（15分钟）

  • 在 MediaShowPage.ets 中编写基础页面结构。
  • 使用 Image($r('app.media.ic_asset')) 显示本地图片（如果没有图片，用系统图标 $r('app.media.startIcon')）。
  • 尝试设置宽高、圆角、边框，观察效果。
    • // 示例代码：本地图片展示 @Entry @Component struct MediaShowPage { build() { Column({ space: 20 }) { Text('资产图片预览') .fontSize(20) .fontWeight(FontWeight.Bold) Image($r('app.media.startIcon')) // 使用自带图标测试 .width(150) .height(150) .borderRadius(20) .border({ width: 2, color: '#0078d4' }) Text('这是我们的设备示例图片') .fontSize(14) .fontColor('#666') } .width('100%') .height('100%') .padding(20) } }

    ✨ 挑战任务： 让图片变成圆形（使用 .borderRadius(75) 或 .borderRadius('50%')）。

    🔧 实训三：加载网络图片（15分钟）

  • 找一张公开的网络图片URL（比如鸿蒙官方logo或免费图片）。
  • 在代码中添加网络图片组件，设置占位图（alt）。
  • 尝试不同的 objectFit 模式（Cover / Contain / Fill），观察图片裁剪效果。
  • 运行到模拟器/真机，确认图片可以显示（网络权限已配好）。
    • // 网络图片示例 Image('https://developer.huawei.com/assets/img/home/harmonyos_logo.png') .width('100%') .height(180) .objectFit(ImageFit.Cover) .alt($r('app.media.startIcon')) // 加载失败时显示默认图 .borderRadius(16)

    🔧 实训四：Video组件 —— 让App播放视频（20分钟）

  • 在页面中添加Video组件，准备一个测试视频URL（可以是公网示例视频，如：https://www.w3schools.com/html/mov_bbb.mp4）。
  • 创建视频控制器 private videoController: VideoController = new VideoController();
  • 添加两个按钮，分别控制播放和暂停，用控制器调用 start() 和 pause()。
  • 监听视频播放状态，在控制台输出日志。
    • @Entry @Component struct VideoDemo { private videoController: VideoController = new VideoController(); @State isPlaying: boolean = false; build() { Column({ space: 15 }) { Video({ src: 'https://www.w3schools.com/html/mov_bbb.mp4', controller: this.videoController }) .width('100%') .height(200) .autoPlay(false) .controls(true) .onStart(() => { this.isPlaying = true; console.info('视频开始播放'); }) .onPause(() => { this.isPlaying = false; console.info('视频暂停'); }) Row({ space: 20 }) { Button('播放') .onClick(() => { this.videoController.start(); }) Button('暂停') .onClick(() => { this.videoController.pause(); }) } } .width('100%') .padding(20) } }

    🎬 如果视频加载慢，可以换成其他小视频源，或者使用本地视频文件（放在rawfile目录）。

    🔧 实训五：综合实战 —— 资产媒体展示页（15分钟）

    场景需求： 为资产管理系统添加“媒体信息”区域，显示资产实物照片 + 操作演示视频，增强应用专业感。

    要求：

    • 上方显示资产图片（网络或本地）
    • 中间显示视频播放器（播放设备使用说明）
    • 下方用Text显示资产名称和简介
    • 整体布局优雅，使用圆角和阴影
  • 组合前面学习的Image和Video组件。
  • 使用Column + Row 进行布局排列。
  • 为图片添加加载失败时的提示（alt属性）。
  • 视频控制可以只显示系统controls，也可以自己加按钮。
  • 运行测试，互相展示成果。
    • // 综合示例（简化版） @Entry @Component struct AssetMediaCard { private videoCtrl: VideoController = new VideoController(); build() { Scroll() { Column({ space: 20 }) { // 资产图片 Image('https://picsum.photos/300/200?random=1') .width('100%') .height(200) .objectFit(ImageFit.Cover) .borderRadius(16) // 资产视频 Video({ src: 'https://www.w3schools.com/html/mov_bbb.mp4', controller: this.videoCtrl }) .width('100%') .height(180) .controls(true) // 文字信息 Column({ space: 8 }) { Text('工业物联网网关') .fontSize(20) .fontWeight(FontWeight.Bold) Text('支持5G通信，实时采集设备数据，提供远程监控能力。') .fontSize(14) .fontColor('#666') } .width('100%') .alignItems(HorizontalAlign.Start) .padding({ left: 8, right: 8 }) } .padding(20) } .backgroundColor('#F5F5F5') } }

    🏆 小组比拼： 看哪个小组的媒体展示页面最漂亮、最像真实参赛作品。每组派代表演示，互相提建议。
