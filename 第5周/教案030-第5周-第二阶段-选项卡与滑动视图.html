🧩 知识讲解 · 选项卡与滑动视图

1. 什么是Tabs组件？—— 像翻书一样切换页面

📖 趣味比喻： Tabs组件就像一本“带标签的书”，每个标签（Tab）对应一个章节（页面），点击标签就翻到那一页。如果用手在屏幕上左右滑动，也能翻页——这就是“滑动视图”。

鸿蒙Tabs组件的核心组成：

  • Tabs： 整个选项卡容器，负责管理所有页面。
  • TabContent： 每一个页面内容，里面可以放任意UI组件（Text、Column、List等）。
  • TabBar： 选项卡栏，显示标签文字和指示器，可以放在顶部或底部。

🎨 效果预览（脑补画面）：

📌 标签栏：[ 资产概览 | 资产列表 | 资产统计 ]

👇 下方区域：点击“资产概览”显示总资产卡片；点击“资产列表”显示资产清单；点击“资产统计”显示图表。

👉 手指在屏幕上左右滑动，也能直接切换内容。

2. Tabs组件的基本写法 —— 三分钟搭建选项卡

@Entry
@Component
struct TabExample {
  build() {
    Column() {
      Tabs() {
        TabContent() {
          Text('这是第一个页面')
            .fontSize(20)
            .width('100%')
            .height('100%')
            .textAlign(TextAlign.Center)
        }
        .tabBar('首页')

        TabContent() {
          Text('这是第二个页面')
            .fontSize(20)
            .width('100%')
            .height('100%')
            .textAlign(TextAlign.Center)
        }
        .tabBar('发现')

        TabContent() {
          Text('这是第三个页面')
            .fontSize(20)
            .width('100%')
            .height('100%')
            .textAlign(TextAlign.Center)
        }
        .tabBar('我的')
      }
      .width('100%')
      .height('100%')
    }
  }
}

💬 代码解析：

  • Tabs { } 是容器，里面每个 TabContent 就是一个页面。
  • .tabBar('标题') 设置标签栏上显示的文字。
  • 默认支持点击切换，但还不支持滑动切换，需要加属性。

3. 开启滑动切换 —— 加一个属性就搞定

👆 关键属性： 在Tabs组件上添加 .scrollable(true)，就能让用户用手指左右滑动切换页面！

Tabs() {
  // ... TabContent内容
}
.scrollable(true)  // 开启滑动切换

此外还可以通过 .barPosition(BarPosition.End) 把标签栏放在底部，适合常用导航App。

4. 自定义选项卡样式 —— 让界面更好看

🎨 文字颜色

.fontColor('#333333')
设置未选中标签的文字颜色

🔵 选中颜色

.selectedFontColor('#0078d4')
设置选中标签的文字颜色

📏 指示器颜色

.indicatorStyle({ color: '#0078d4', height: 3 })
设置底部指示条

@Component
struct AssetListPage {
  @State assets: string[] = ['笔记本电脑 (45台)', '投影仪 (12台)', '实验设备 (99台)', '服务器 (8台)']
  build() {
    Column() {
      Text('📋 资产清单')
        .fontSize(24)
        .fontWeight(FontWeight.Bold)
        .margin({ top: 20, bottom: 15 })
      List() {
        ForEach(this.assets, (item: string) => {
          ListItem() {
            Row() {
              Text(item)
                .fontSize(18)
                .padding(12)
            }
            .width('100%')
            .backgroundColor('#F8F9FA')
            .borderRadius(12)
            .margin({ bottom: 8 })
          }
        })
      }
      .width('100%')
      .layoutWeight(1)
      .padding({ left: 16, right: 16 })
    }
    .width('100%')
    .height('100%')
    .backgroundColor('#FFFFFF')
  }
}
