🧩 知识讲解 · 为什么数据可视化是大赛加分项

1. 数字 vs 图表 —— 一眼看懂的力量

📉 趣味案例： 给你两组数字：
“电脑类:45台， 显示器:32台， 键盘:67个……” 和一张柱状图，哪个能让你3秒知道最多的是什么？

评委视角： 参赛作品如果内嵌漂亮的数据报表，不仅展示技术能力，还体现“产品思维”——让用户决策更容易。

鸿蒙应用可以用Canvas组件自绘或集成MPChart（OpenHarmony图表库）。我们这节课以轻量级方式：用Canvas绘制柱状图，不依赖第三方包，便于理解原理。

2. 鸿蒙图表实现方案对比

🎨 Canvas自绘

轻量、定制灵活，适合简单图表（柱状图、折线图），无外部依赖。

📦 MPChart

功能强大（饼图、雷达图等），但需要引入三方库，有一定学习成本。

📊 自定义组件

复用性高，适合项目中多次使用图表。

✅ 本堂课选择：Canvas自绘柱状图 + 简单饼图演示，让学生掌握最硬核的绘图逻辑。

3. 数据聚合SQL —— 为图表准备素材

-- 按资产类别统计数量（示例） SELECT asset_type, COUNT(*) as count FROM assets GROUP BY asset_type; -- 按月统计资产增加趋势（假设有create_time字段） SELECT strftime('%Y-%m', create_time) as month, COUNT(*) FROM assets GROUP BY month;

我们实训中将使用模拟的资产数据，通过代码聚合出“类别-数量”映射，然后绘制柱状图。

    🧪 实训任务二：使用Canvas绘制柱状图（25分钟）

  • 在build()中添加Canvas组件，设置宽高（例如宽度:`this.canvasWidth=350`, 高度:`200`）。
  • 通过 `onReady` 事件获取CanvasRenderingContext2D，并编写绘制函数 `drawBarChart`。
  • 动态计算每个柱子的宽度、间距、最高柱子比例，适配数据。
  • 给Canvas添加绘制逻辑，同时支持屏幕旋转或数据更新时重绘。
  • 核心代码示范（关键部分）：
    • private settings: RenderingContextSettings = new RenderingContextSettings(true); private context: CanvasRenderingContext2D = new CanvasRenderingContext2D(this.settings); build() { Column() { Canvas(this.context) .width(360) .height(240) .backgroundColor('#F5F5F5') .onReady(() => { this.drawBarChart(); }) } } drawBarChart() { if (!this.context) return; const ctx = this.context; const width = 360, height = 240; ctx.clearRect(0, 0, width, height); const data = this.statData; if (data.length === 0) return; const maxCount = Math.max(...data.map(item => item.count)); const barWidth = (width - 60) / data.length - 6; const startX = 40; for (let i = 0; i < data.length; i++) { const barHeight = (data[i].count / maxCount) * (height - 60); const x = startX + i * (barWidth + 8); const y = height - 30 - barHeight; ctx.fillStyle = '#0078D4'; ctx.fillRect(x, y, barWidth, barHeight); ctx.fillStyle = '#333'; ctx.font = '12px sans-serif'; ctx.fillText(data[i].type, x + 4, height - 8); ctx.fillStyle = '#111'; ctx.fillText(data[i].count.toString(), x + barWidth / 4, y - 4); } }

    📐 柱状图已经动起来了！学生可以尝试修改颜色、加标题、显示数值标签；教师巡导，解决坐标系偏移问题。

    🧪 实训任务三：添加饼图展示占比（进阶·20分钟）

  • 同样使用Canvas，在右侧或下方区域绘制简易饼图。
  • 原理：计算每种资产所占角度，使用 `ctx.arc` 和 `ctx.fill` 绘制扇形。
  • 简单版本：绘制一个圆形，根据比例填充不同颜色块。
  • 教师可提供“饼图绘制函数”代码片段，学生实现颜色映射。
    • drawPieChart() { const ctx = this.context; const total = this.statData.reduce((sum, item) => sum + item.count, 0); let startAngle = -Math.PI / 2; const colors = ['#FF6B6B', '#4ECDC4', '#FFE66D', '#1A535C', '#FF9F1C']; for (let i = 0; i < this.statData.length; i++) { const angle = (this.statData[i].count / total) * 2 * Math.PI; ctx.beginPath(); ctx.fillStyle = colors[i % colors.length]; ctx.moveTo(180, 120); ctx.arc(180, 120, 80, startAngle, startAngle + angle); ctx.closePath(); ctx.fill(); startAngle += angle; } }
