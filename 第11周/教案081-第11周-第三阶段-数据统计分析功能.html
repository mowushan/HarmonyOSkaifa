🧩 知识讲解 · 数据分析三剑客

1. 为什么需要统计分析？—— 从“记录”到“洞察”

📊 我们的资产管理系统已经能录入、存储、编辑、删除。但光有数据列表还不够！一个好的系统应该能回答：

  • 🔹 我们一共有多少件资产？总价值多少？
  • 🔹 哪个分类的资产数量最多？哪个最值钱？
  • 🔹 近一周新增了多少设备？
  • 🔹 不同位置的资产分布如何？

这些问题的答案，就是统计分析的价值 —— 让数据辅助决策！

2. SQL聚合函数 —— 数据统计的核心武器

COUNT(*) — 统计行数
SELECT COUNT(*) FROM assets;
SUM(price) — 求和
SELECT SUM(price) FROM assets;
AVG(price) — 平均值
SELECT AVG(price) FROM assets;
MAX(price) — 最大值
SELECT MAX(price) FROM assets;
MIN(price) — 最小值
SELECT MIN(price) FROM assets;
GROUP BY — 分组统计
SELECT category, COUNT(*) FROM assets GROUP BY category;
-- 示例：按资产类型统计数量和总价值
SELECT category,
       COUNT(*) as count,
       SUM(price) as total_value,
       AVG(price) as avg_price
FROM assets
GROUP BY category
ORDER BY total_value DESC;

3. 时间筛选 & 条件统计

📅 通过日期字段可以统计“本月新增”、“近7天入库”等动态数据。

-- 统计本周入库资产数量（假设日期格式 yyyy-MM-dd）
SELECT COUNT(*) 
FROM assets 
WHERE date(create_time) >= date('now', '-7 days');

🧪 实训一：编写统计工具类（核心逻辑）

  • 创建 StatisticsHelper 类，封装所有统计查询方法。
  • 实现 getTotalCount()：查询资产总数量。
  • 实现 getTotalValue()：查询资产总价值。
  • 实现 getCategoryStatistics()：返回每个分类的名称、数量、总价值。
  • 实现 getRecentAddedCount(days)：查询最近N天新增资产数。
    • public class StatisticsHelper {
    private DatabaseHelper dbHelper;
    
    public int getTotalCount() {
        SQLiteDatabase db = dbHelper.getReadableDatabase();
        Cursor cursor = db.rawQuery("SELECT COUNT(*) FROM assets", null);
        cursor.moveToFirst();
        int count = cursor.getInt(0);
        cursor.close();
        return count;
    }
    
    public double getTotalValue() {
        SQLiteDatabase db = dbHelper.getReadableDatabase();
        Cursor cursor = db.rawQuery("SELECT SUM(price) FROM assets", null);
        cursor.moveToFirst();
        double total = cursor.getDouble(0);
        cursor.close();
        return total;
    }
}

    🧪 实训二：分类统计（GROUP BY）

  • 编写方法 getCategoryStatsList()，返回一个包含分类名、数量、总价值的列表。
  • 遍历Cursor并封装成对象或Map，供界面展示。
    • public List<CategoryStat> getCategoryStatsList() {
    List<CategoryStat> list = new ArrayList<>();
    String sql = "SELECT category, COUNT(*) as cnt, SUM(price) as total "
               + "FROM assets GROUP BY category ORDER BY total DESC";
    Cursor cursor = db.rawQuery(sql, null);
    while (cursor.moveToNext()) {
        String category = cursor.getString(0);
        int count = cursor.getInt(1);
        double total = cursor.getDouble(2);
        list.add(new CategoryStat(category, count, total));
    }
    cursor.close();
    return list;
}
