🧩 知识讲解 · Preferences轻量级存储

1. Preferences是什么？—— 应用的小便签本

📝 趣味比喻： Preferences就像App的“小便签本”，每次用户做了设置，就在便签本上记下来。下次打开App，翻一翻便签本，就能恢复之前的设置。

适合存储的数据类型：

  • ⚙️ 用户偏好设置：是否开启通知、夜间模式开关、字体大小
  • 👤 用户信息：昵称、头像路径、最后一次登录时间
  • 🎨 主题颜色：用户选择的主题色
  • 📊 统计数据：应用启动次数、上次使用时间

❌ 不适合存储： 大量数据、复杂数据结构、需要频繁修改的数据（建议用数据库）。

2. Preferences核心API —— 三步搞定存储

📖 第一步：获取实例

import preferences from '@ohos.data.preferences';

let pref = await preferences.getPreferences(this.context, "my_settings");

参数1：上下文；参数2：文件名（会生成my_settings文件）

✍️ 第二步：写入数据

await pref.put("username", "张三");
await pref.put("isDarkMode", true);
await pref.flush();  // 必须调用flush才能持久化

put()写入内存，flush()刷到磁盘

🔍 第三步：读取数据

let username = await pref.get("username", "默认值");
let isDarkMode = await pref.get("isDarkMode", false);

get(键, 默认值) 如果键不存在，返回默认值

3. 完整代码示例 —— 设置页面的实现

📌 场景： 我们要做一个“偏好设置”页面，包含：用户名输入框、夜间模式开关、通知开关。用户修改后自动保存，下次打开应用自动恢复。

// 完整代码：SettingsPage.ets
import preferences from '@ohos.data.preferences';
import router from '@ohos.router';

@Entry
@Component
struct SettingsPage {
    @State username: string = '';
    @State isDarkMode: boolean = false;
    @State isNotificationEnabled: boolean = true;
    private pref: preferences.Preferences | null = null;

    async aboutToAppear() {
        // 页面加载时读取保存的设置
        await this.loadSettings();
    }

    async loadSettings() {
        try {
            // 1. 获取Preferences实例
            this.pref = await preferences.getPreferences(getContext(this), "user_settings");
            
            // 2. 读取数据（第二个参数是默认值）
            this.username = await this.pref.get("username", "游客");
            this.isDarkMode = await this.pref.get("isDarkMode", false);
            this.isNotificationEnabled = await this.pref.get("isNotificationEnabled", true);
        } catch (error) {
            console.error("加载设置失败: " + error);
        }
    }

    async saveSettings() {
        if (!this.pref) return;
        try {
            // 写入数据
            await this.pref.put("username", this.username);
            await this.pref.put("isDarkMode", this.isDarkMode);
            await this.pref.put("isNotificationEnabled", this.isNotificationEnabled);
            
            // 必须调用flush才能持久化保存到磁盘
            await this.pref.flush();
            
            console.info("设置保存成功");
        } catch (error) {
            console.error("保存设置失败: " + error);
        }
    }

    build() {
        Column() {
            Text("⚙️ 偏好设置")
                .fontSize(24)
                .fontWeight(FontWeight.Bold)
                .margin({ top: 20, bottom: 20 })

            // 用户名设置
            Row() {
                Text("用户名：")
                    .width(100)
                TextInput({ placeholder: "请输入用户名", text: this.username })
                    .layoutWeight(1)
                    .onChange((value: string) => {
                        this.username = value;
                        this.saveSettings();  // 实时保存
                    })
            }
            .width('100%')
            .padding(12)

            // 夜间模式开关
            Row() {
                Text("🌙 夜间模式")
                    .width(100)
                Toggle({ type: ToggleType.Switch, isOn: this.isDarkMode })
                    .onChange((isOn: boolean) => {
                        this.isDarkMode = isOn;
                        this.saveSettings();
                    })
            }
            .width('100%')
            .padding(12)

            // 通知开关
            Row() {
                Text("🔔 接收通知")
                    .width(100)
                Toggle({ type: ToggleType.Switch, isOn: this.isNotificationEnabled })
                    .onChange((isOn: boolean) => {
                        this.isNotificationEnabled = isOn;
                        this.saveSettings();
                    })
            }
            .width('100%')
            .padding(12)

            Button("查看保存的数据")
                .width('80%')
                .margin({ top: 30 })
                .onClick(() => {
                    console.info(`用户名: ${this.username}, 夜间模式: ${this.isDarkMode}, 通知: ${this.isNotificationEnabled}`);
                    alert(`当前设置:\n用户名: ${this.username}\n夜间模式: ${this.isDarkMode}\n通知: ${this.isNotificationEnabled}`);
                })
        }
        .width('100%')
        .height('100%')
        .padding(20)
    }
}

💡 关键点解释：

  • aboutToAppear()：页面加载前自动调用，用来读取之前保存的设置。
  • saveSettings()：每次用户修改设置时调用，实时保存。
  • flush()：非常关键！不调用flush，数据只存在内存中，应用退出就丢失。

  • 新建工具文件：在common目录下创建 PreferencesUtil.ets
  • 编写工具类，封装Preferences的初始化、保存、读取方法： 
    import preferences from '@ohos.data.preferences';

export class PreferencesUtil {
    private static instance: PreferencesUtil;
    private pref: preferences.Preferences | null = null;

    static getInstance(): PreferencesUtil {
        if (!PreferencesUtil.instance) {
            PreferencesUtil.instance = new PreferencesUtil();
        }
        return PreferencesUtil.instance;
    }

    async init(context: Context) {
        this.pref = await preferences.getPreferences(context, "my_preferences");
    }

    async put(key: string, value: string | number | boolean) {
        if (!this.pref) return;
        await this.pref.put(key, value);
        await this.pref.flush();
    }

    async get(key: string, defaultValue: string | number | boolean) {
        if (!this.pref) return defaultValue;
        return await this.pref.get(key, defaultValue);
    }
}
