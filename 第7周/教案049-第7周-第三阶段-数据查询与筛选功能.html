🧩 知识讲解 · 数据库查询魔法

1. SQL查询基础 —— 像“百度一下”一样查资产

🗄️ 趣味引入： 假设你的资产数据库是一个巨大的“仓库”，里面堆满了设备。你想找“笔记本电脑”，如果没有查询功能，你得翻遍整个仓库！SQL就是你的“仓库管理员”，你喊一声，它帮你快速找到。

最常用的查询命令模板：

-- 查询所有资产 SELECT * FROM asset_table; -- 条件查询：只查名称包含“电脑”的资产 SELECT * FROM asset_table WHERE asset_name LIKE '%电脑%'; -- 多条件查询：分类是“电子设备”且购入日期在2025年之后 SELECT * FROM asset_table WHERE category = '电子设备' AND purchase_date >= '2025-01-01';

📖 关键词解释：

  • SELECT：选择要查询的列，* 表示所有列
  • FROM：从哪张表里查
  • WHERE：设置筛选条件（等于、大于、模糊匹配）
  • LIKE：模糊匹配，% 代表任意多个字符

2. 鸿蒙中执行查询 · 三步走

🔧 第一步：构建谓词

使用 dataSharePredicates 对象来封装查询条件，相当于把SQL语句翻译成鸿蒙能理解的对象。

⚙️ 第二步：执行查询

调用 store.query() 方法，传入谓词，拿到结果集 ResultSet。

📋 第三步：遍历结果

使用 while(resultSet.goToNextRow()) 循环取出每一行数据，存入列表刷新UI。

// 示例：按资产名称模糊查询 let predicates = new dataSharePredicates.DataSharePredicates(); predicates.like("asset_name", `%${searchText}%`); let resultSet = await this.store.query(predicates, ["asset_id", "asset_name", "category", "purchase_date"]); let assets = []; while (resultSet.goToNextRow()) { let asset = { id: resultSet.getLong(resultSet.getColumnIndex("asset_id")), name: resultSet.getString(resultSet.getColumnIndex("asset_name")), category: resultSet.getString(resultSet.getColumnIndex("category")), date: resultSet.getString(resultSet.getColumnIndex("purchase_date")) }; assets.push(asset); } resultSet.close(); // 记得关闭结果集释放资源

🧪 实训任务一：添加搜索框组件（15分钟）

  • 修改页面布局： 在列表上方添加 TextInput 组件，用于输入搜索关键词。
  • 绑定状态变量： 定义 @State searchText: string = '';，并绑定到输入框。
  • 添加搜索按钮或实时监听： 使用 .onChange 事件，当输入内容变化时触发查询。
    • // 在 AssetList.ets 中添加以下代码（简略示例） @State searchText: string = ''; @State assetList: AssetItem[] = []; build() { Column() { // 搜索栏 Row() { TextInput({ placeholder: "输入资产名称搜索", text: this.searchText }) .onChange((value: string) => { this.searchText = value; this.loadAssetsWithSearch(); // 调用带筛选的加载方法 }) .layoutWeight(1) .margin({ right: 10 }) Button("搜索") .onClick(() => { this.loadAssetsWithSearch(); }) } .padding(12) .width('100%') // 资产列表（原来的 List 组件） List() { ... } } }

    🧪 实训任务二：实现带条件查询的 loadAssetsWithSearch()（25分钟）

  • 复制原有加载方法： 将之前写好的 loadAssets() 复制一份，重命名为 loadAssetsWithSearch()。
  • 构建谓词： 如果 searchText 不为空，添加 like 条件；否则查询全部。
  • 执行查询并刷新列表： 将结果赋值给 assetList。
  • 添加打印日志： 打印最终生成的SQL条件，方便调试。
    • async loadAssetsWithSearch() { let predicates = new dataSharePredicates.DataSharePredicates(); // 动态添加筛选条件 if (this.searchText && this.searchText.trim().length > 0) { predicates.like("asset_name", `%${this.searchText}%`); console.info(`查询条件: asset_name LIKE '%${this.searchText}%'`); } else { console.info("查询全部资产"); } // 按购入日期倒序排列（最新的在前） predicates.orderByDesc("purchase_date"); try { let resultSet = await this.store.query(predicates, [ "asset_id", "asset_name", "category", "purchase_date", "unit_price", "quantity" ]); let newList: AssetItem[] = []; while (resultSet.goToNextRow()) { let item = new AssetItem(); item.id = resultSet.getLong(resultSet.getColumnIndex("asset_id")); item.name = resultSet.getString(resultSet.getColumnIndex("asset_name")); item.category = resultSet.getString(resultSet.getColumnIndex("category")); item.date = resultSet.getString(resultSet.getColumnIndex("purchase_date")); item.price = resultSet.getDouble(resultSet.getColumnIndex("unit_price")); item.qty = resultSet.getLong(resultSet.getColumnIndex("quantity")); newList.push(item); } resultSet.close(); this.assetList = newList; } catch (err) { console.error(`查询失败: ${err.message}`); } }

    🐞 常见Bug与解决：

    • ❌ 查询不到结果 → 检查数据库中确实有匹配的数据，检查字段名是否拼写正确（asset_name 还是 name？）
    • ❌ 应用崩溃 → 忘记 resultSet.close() 或未捕获异常
    • ❌ 输入框输入中文不触发 → 确认 .onChange 事件绑定正确，打印 value 看是否变化
